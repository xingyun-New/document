## 💡 如何添加题目文字

### 方案一：手动编辑（推荐）
1. 打开Markdown文件
2. 查看对应页面的图片
3. 在图片上方手动输入题目内容

示例格式：
```markdown
## 第 1 页

**1. 选择题**

(A) 选项A
(B) 选项B  
(C) 选项C
(D) 选项D

![图片](images/page1_img1.jpeg)
```

### 方案二：使用在线OCR工具
推荐工具：
1. **百度OCR** - https://cloud.baidu.com/product/ocr
2. **腾讯OCR** - https://cloud.tencent.com/product/ocr
3. **天若OCR** - 桌面截图识别工具

步骤：
1. 打开images文件夹中的图片
2. 使用OCR工具识别文字
3. 复制识别结果到Markdown文件

### 方案三：批量处理脚本
如果需要批量OCR识别，可以：
1. 使用Python + Tesseract OCR
2. 使用云端OCR API (百度/腾讯)
3. 联系我帮助编写专用脚本
