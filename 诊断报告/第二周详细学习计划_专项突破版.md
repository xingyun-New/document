### 遗传题必背公式

```
【基因型与表现型】
AA×AA → 全AA（全显性）
AA×Aa → AA:Aa = 1:1（全显性）
AA×aa → 全Aa（全显性）
Aa×Aa → AA:Aa:aa = 1:2:1（显:隐 = 3:1）
Aa×aa → Aa:aa = 1:1（显:隐 = 1:1）
aa×aa → 全aa（全隐性）

【遗传图解格式】
亲代：    ♂AA × ♀aa
           ↓     ↓
配子：     A      a
            \   /
子代：      Aa（全为杂合子）
```
