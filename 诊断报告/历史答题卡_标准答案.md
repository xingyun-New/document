## 第Ⅱ卷 非选择题答案（50分）

### 26题（16分）

#### (1) 中国沦为半殖民地半封建社会的标志性事件（6分）

```
① 《辛丑条约》签订（1901年）——标志中国完全沦为半殖民地半封建社会

② 《马关条约》签订（1895年）——大大加深了中国半殖民地化程度

③ 《南京条约》签订（1842年）——中国开始沦为半殖民地半封建社会
```

#### (2) 近代中国人民反侵略斗争的特点（3分）

```
① 顽强抵抗，前赴后继，涌现出无数英雄人物

② 斗争从自发到自觉，逐步形成有组织的革命运动

③ 反侵略与反封建相结合，既反对外国侵略，又反对本国封建统治
```

#### (3)① 近代中国革命运动的趋势（5分）

```
趋势：从旧民主主义革命向新民主主义革命转变

史实说明：
旧民主主义革命时期（1840-1919）：太平天国运动、戊戌变法、辛亥革命等，
由农民阶级和资产阶级领导，未能完成反帝反封建任务。

新民主主义革命时期（1919-1949）：五四运动后，中国共产党领导，以工人阶级
为领导，最终取得了革命胜利，建立了新中国。
```

#### (3)② 补充说明（2分）

```
革命主体从农民、资产阶级转向工人阶级和共产党领导；革命目标更加明确；
革命理论从改良主义、三民主义发展到马克思主义。
```

### 27题（17分）

#### (1)① 近代民族工业发展的特点（2分）

```
① 近代民族工业在夹缝中艰难发展

② 受到外国资本主义和本国封建势力的双重压迫，呈现"先天不足，后天畸形"的特点
```

#### (1)② 发展阶段（补充）

```
第一阶段（1927年前）：民族工业初步发展

第二阶段（一战期间）：黄金时期，获得较快发展

第三阶段（1929年后）：受世界经济危机影响，再次受挫
```

#### (2) 影响因素与历史贡献（3分）

```
影响因素：
① 外国资本主义的侵略和经济掠夺
② 本国封建势力的压制
③ 战争与社会动荡
④ 缺乏资金、技术和市场

历史贡献：
① 推动了中国近代化进程，培养了近代产业工人
② 为中国革命提供了物质基础和阶级基础
③ 促进了中国社会结构的变化，推动了思想解放
```

#### (3)① 近代中国工业发展的规律特征（2分）

```
① 在战争中受挫，在和平时期发展

② 具有明显的阶段性和波动性，始终受制于半殖民地半封建社会的性质
```

#### (3)② 革命传统教育（3分）

```
① 近代民族资本家在艰苦条件下坚持实业救国，体现了爱国精神

② 中国人民不屈不挠，在半殖民地半封建社会中顽强抗争

③ 只有在中国共产党领导下，走社会主义道路，才能实现民族独立和国家富强
```

### 28题（17分）

#### (1) 李大钊的贡献（6分）+ 精神（3分）

```
贡献：
① 传播马克思主义（1917年以来）：多次到天津宣传马克思主义，影响大批爱国青年

② 建立早期组织（1920年）：指导成立马克思主义研究会

③ 培养革命骨干：与陈独秀南北呼应，为中国共产党的创建发出先声

④ 建立地方党组织（1924年）：协助建立中国共产党天津地方执行委员会

精神：
① 献身精神：为共产主义事业奋斗终身，不怕牺牲

② 开拓精神：率先传播马克思主义，开创中国革命新道路

③ 奉献精神：全心全意为人民服务，培养革命后继人才
```

#### (2) 搜集革命家故事的方法（4分）

```
① 阅读革命历史书籍、人物传记

② 参观革命纪念馆、博物馆、烈士陵园

③ 采访老党员、老革命、老战士

④ 观看革命历史题材的影视作品、纪录片
```

#### 周恩来的革命故事（4分）

```
故事名称：周恩来在南开学校的革命觉醒

主要贡献：
① "为中华之崛起而读书"：在南开学校时期就树立了远大志向，立志救国救民

② 参与五四运动：作为学生领袖，积极组织天津学生运动，创办《觉悟》杂志

③ 赴法勤工俭学：1920年赴法国勤工俭学，接触并接受了马克思主义

④ 建立共产党组织：在欧洲建立中国共产党旅欧支部，成为早期共产党人

意义：周恩来从青年时代就投身革命，为中国革命和建设事业奉献了毕生精力，
是我们学习的光辉榜样。
```
