#### 📖 辅助线口诀（必背！）

```
遇到中点做延长，倍长中线显神通
三角形中两中点，连接则为中位线
角平分线平行线，等腰三角常出现
遇到直角做斜边，弦中线取一半
直角三角形中线，斜边中点连直角
```

---

#### 📖 辅助线类型一：倍长中线

**适用条件：** 已知中线，需要构造全等

**方法：** 延长中线一倍

<img src="geo_images/day12_geo_01.jpg" width="280">

> **图示说明**：△ABC中，D是BC中点，延长AD到E使DE = AD

**构造原理：**
- 此时：△ABD ≌ △ECD（SAS）
- 因为：AD = DE，BD = CD，∠ADB = ∠EDC（对顶角）

**📝 例题：** 在△ABC中，D是BC中点，AD是中线，且AD < ½BC。
延长AD到E使DE = AD，求证：∠BAC > 90°

---

#### 📖 辅助线类型二：作垂线

**适用条件：** 需要用到直角三角形性质、勾股定理

**方法：** 从顶点向对边作垂线

```
       A
      /|\
     / |h\
    /  |  \
   B———H———C

作AH ⊥ BC于H
```

**作用：**
1. 构造直角三角形
2. 运用勾股定理
3. 计算面积

---

#### 📖 辅助线类型三：作平行线

**适用条件：** 需要用到平行线性质

**方法：** 过某点作平行线

**作用：**
1. 等量转换（同位角、内错角相等）
2. 构造平行四边形
3. 截长补短

**📝 例题：** 如图，在△ABC中，∠BAC的平分线交BC于D，过D作DE∥AC交AB于E

```
       A
      /|\
     / | \
    E——D  \
   /   |   \
  B————+————C
```

则：∠AED = ∠DAC = ∠BAD（平行线+角平分线）
所以：AE = DE（等角对等边）

---

#### 📖 辅助线类型四：连接关键点

**适用条件：** 题目给出特殊点（中点、角平分线上的点等）

**方法：** 连接题目中的关键点

**常见连接：**
1. 连接对角线顶点
2. 连接中点
3. 连接角平分线上的点与边

---

#### 📖 辅助线类型五：截长补短

**适用条件：** 证明线段关系

**方法一：截长**
在长线段上截取等于短线段的一段

**方法二：补短**
将短线段延长到等于长线段

**【例题】** 在等腰△ABC中，AB = AC，D是BC上一点，且∠BAD = 30°。
在AB上取点E，使AE = AD，求证：DE = BD


### ⏰ 10:30-12:00 | 生物·生态系统综合（90分钟）

#### 📖 知识点一：生态系统的组成

```
生态系统 = 生物部分 + 非生物部分

生物部分：
├── 生产者（绿色植物）——能进行光合作用
├── 消费者（动物）——直接或间接以植物为食
└── 分解者（细菌、真菌）——分解有机物

非生物部分：阳光、空气、水、土壤等
```

---

#### 📖 知识点二：食物链

**定义：** 在生态系统中，不同生物之间由于吃与被吃的关系形成的链状结构。

**书写规则：**
```
生产者 → 初级消费者 → 次级消费者 → ...

例：草 → 兔 → 狐狸 → 狼
    ↑        ↑       ↑      ↑
  生产者  初级    次级   三级
         消费者  消费者  消费者
```

**🔥 注意事项：**
1. 食物链起点必须是生产者
2. 箭头表示能量流动方向（被吃→吃）
3. 分解者不参与食物链
4. 营养级 = 位置（生产者是第一营养级）

---

#### 📖 知识点三：食物网

**定义：** 多条食物链相互交织形成的网状结构

```
        鹰
       ↗ ↖
     蛇   猫头鹰
    ↗  ↘   ↗
  青蛙  ←  鼠
   ↑      ↗
   昆虫 ← 
   ↑
  植物
```

**分析方法：**
1. 找出所有食物链
2. 确定某生物的营养级（可能有多个）
3. 分析某生物数量变化对其他生物的影响

---

#### 📖 知识点四：能量流动

**特点：**
1. **单向流动：** 只能从低营养级流向高营养级
2. **逐级递减：** 传递效率约10%-20%

```
太阳能 → 生产者(100%) → 初级消费者(10-20%) → 次级消费者(1-4%)
```

**能量计算：**
```
例：如果生产者固定的能量为10000千焦
    传递效率为10%
    则：初级消费者获得 = 10000 × 10% = 1000千焦
        次级消费者获得 = 1000 × 10% = 100千焦
```

**📌 考试技巧：**
- 求"最多"能量 → 用最高效率20%
- 求"最少"能量 → 用最低效率10%

---

#### 📖 知识点五：生态平衡

**定义：** 生态系统中各种生物的数量和所占比例保持相对稳定的状态

**自我调节能力：**
- 生态系统具有一定的自我调节能力
- 生物种类越多，营养结构越复杂，自我调节能力越强

### ⏰ 17:00-18:00 | 道法·辨析题答题技巧（60分钟）

#### 📖 辨析题标准答题格式

**答题结构（四步走）：**

```
【第一步：判断】
这个观点是正确的/错误的/片面的。

【第二步：分析正确部分】
（如果观点是正确的或片面的）
__________是正确的，因为__________。

【第三步：分析错误部分】
（如果观点是错误的或片面的）
但是__________是错误的/不全面的，因为__________。

【第四步：总结】
因此，我们应该__________。
```

---

#### 📝 辨析题示例

**【例题】** 辨析：有人认为"网络是把双刃剑"。

**答案：**
```
（1）这个观点是正确的。

（2）一方面，网络有积极作用。网络可以帮助我们获取信息、
学习知识、拓宽视野，还可以方便我们与他人交流。

（3）另一方面，网络也有消极影响。网络上存在虚假信息、
不良内容，沉迷网络会影响学习和健康，个人隐私也可能泄露。

（4）因此，我们应该正确使用网络，发挥网络的积极作用，
避免网络的消极影响，做网络的主人而不是网络的奴隶。
```

---

#### 📝 常见辨析题类型

**类型一：完全正确型**
- 判断：正确
- 只需分析为什么正确

**类型二：完全错误型**
- 判断：错误
- 只需分析为什么错误

**类型三：片面型（最常见）**
- 判断：片面/不全面
- 既要说正确的部分，也要指出不足
