#### 知识点：方程应用题

**学习流程：**

**第一步：掌握解题步骤（15分钟）**

**解题步骤（5步法）：**

**1. 审题**
- 仔细读题，理解题意
- 找出已知量和未知量
- 找出等量关系

**2. 设未知数**
- 设x为所求量（直接设元）
- 或设x为中间量（间接设元）
- 写清楚单位

**3. 列方程**
- 根据等量关系列方程
- 注意单位统一

**4. 解方程**
- 按照解方程步骤求解
- 计算准确

**5. 答**
- 写出答语
- 带单位
- 检验是否符合实际

**记忆口诀：**
> 审设列解答，五步不能少  
> 审题要仔细，设元要清楚  
> 列式找关系，解题要准确  
> 最后别忘答，单位要写上

---

**第二步：掌握常见应用题类型（35分钟）**

**类型1：行程问题（12分钟）**

**基本公式：** 路程 = 速度 × 时间

**常见问题类型：**

**①相遇问题**
- 关键：两人行驶的路程之和 = 总路程
- 公式：(速度1 + 速度2) × 时间 = 总路程

**例题1：**
```
甲乙两人相距100千米，同时相向而行，甲速度20千米/时，
乙速度30千米/时，多久相遇？

解：设x小时后相遇
根据题意：20x + 30x = 100
        50x = 100
         x = 2
答：2小时后相遇。
```

---

**②追及问题**
- 关键：快者行驶的路程 - 慢者行驶的路程 = 相差路程
- 公式：(速度1 - 速度2) × 时间 = 相差路程

**例题2：**
```
甲在乙前面10千米，甲速度15千米/时，乙速度20千米/时，
乙多久追上甲？

解：设x小时后乙追上甲
根据题意：20x - 15x = 10
        5x = 10
         x = 2
答：2小时后乙追上甲。
```

---

**③往返问题**
- 关键：去时路程 = 回来路程

**例题3：**
```
小明从家到学校用了0.5小时，速度4千米/时，回来时用了
0.4小时，求回来时的速度。

解：设回来时速度为x千米/时
根据题意：4×0.5 = 0.4x
        2 = 0.4x
        x = 5
答：回来时速度为5千米/时。
```

---

**类型2：工程问题（12分钟）**

**基本公式：** 工作量 = 工作效率 × 工作时间

**解题技巧：** 把总工作量看作"1"

**常见问题类型：**

**①合作问题**

**例题4：**
```
甲单独完成需10天，乙单独完成需15天，两人合作几天完成？

解：设x天完成
甲的效率：1/10
乙的效率：1/15
根据题意：(1/10 + 1/15)x = 1
        (3/30 + 2/30)x = 1
        5/30 · x = 1
        x = 6
答：6天完成。
```

---

**②先后工作问题**

**例题5：**
```
一项工程，甲单独做需12天，乙单独做需18天。甲先做3天，
剩下的两人合作，还需几天完成？

解：设还需x天完成
甲的效率：1/12
乙的效率：1/18
根据题意：1/12 × 3 + (1/12 + 1/18)x = 1
        1/4 + 5/36 · x = 1
        5/36 · x = 3/4
        x = 27/5 = 5.4
答：还需5.4天完成。
```

---

**类型3：利润问题（11分钟）**

**基本公式：**
- 利润 = 售价 - 成本
- 利润率 = 利润 / 成本
- 售价 = 成本 × (1 + 利润率)

**例题6：**
```
某商品成本80元，商家要获得20%的利润，售价应定为多少？

解：设售价为x元
根据题意：x - 80 = 80 × 20%
        x - 80 = 16
        x = 96
答：售价应定为96元。
```

---

**例题7：**
```
某商品原价100元，打8折后售价为80元，仍获利20%，求成本。

解：设成本为x元
根据题意：80 - x = 20% · x
        80 = x + 0.2x
        80 = 1.2x
        x = 200/3 ≈ 66.67
答：成本约为66.67元。
```

---

**第三步：总结等量关系（10分钟）**

**常见等量关系：**

**行程问题：**
- 相遇：路程1 + 路程2 = 总路程
- 追及：路程1 - 路程2 = 相差路程
- 往返：去时路程 = 回来路程

**工程问题：**
- 工作量 = 工作效率之和 × 时间
- 总工作量 = 1

**利润问题：**
- 售价 = 成本 + 利润
- 利润 = 成本 × 利润率

**记忆：** 每种类型至少掌握一个典型例题！

### ⏰ 11:30-12:00 | 生物重点突破（30分钟）

> 🎯 **重点原因：** 试卷分析显示，生态系统是第3优先级，预计丢分4-6分。

#### 知识点：生态系统的组成

**学习流程：**

**第一步：理解生态系统的概念（10分钟）**

**生态系统的定义：**
在一定的空间范围内，生物与环境所形成的统一整体。

**生态系统的组成：**

```
      生态系统
         |
    _____|_____
   |           |
非生物成分   生物成分
   |           |
阳光、空气    生产者
水、温度      消费者
土壤等        分解者
```

---

**一、非生物成分**

**包括：**
- 阳光（能量来源）
- 空气（氧气、二氧化碳）
- 水
- 温度
- 土壤
- 无机盐

**作用：** 为生物提供物质和能量

---

**二、生物成分**

**1. 生产者**
- **定义：** 能进行光合作用，制造有机物的生物
- **主要是：** 绿色植物
- **作用：** 制造有机物，是生态系统的基础

**例子：** 草、树、农作物、藻类

---

**2. 消费者**
- **定义：** 不能制造有机物，直接或间接以植物为食的生物
- **主要是：** 动物
- **分类：**
  - 初级消费者（植食性动物）：兔子、牛、羊
  - 次级消费者（肉食性动物）：狼、鹰
  - 三级消费者：更高级的肉食动物

**例子：** 兔子、青蛙、蛇、鹰

---

**3. 分解者**
- **定义：** 能分解有机物为无机物的生物
- **主要是：** 细菌和真菌
- **作用：** 分解有机物，使物质循环

**例子：** 细菌、真菌、蚯蚓

---

**三者关系：**

```
  阳光能量
     ↓
  生产者（植物）制造有机物
     ↓
  消费者（动物）利用有机物
     ↓
  生产者和消费者的遗体、排泄物
     ↓
  分解者（细菌、真菌）分解
     ↓
  无机物回到环境
     ↓
  生产者重新利用
```

**记忆口诀：**
> 生产者制造有机物，消费者吃它来生存  
> 分解者分解遗体物，无机物又回环境

### ⏰ 15:15-16:15 | 生物深度学习（60分钟）

> 📝 **重点：** 食物链和能量流动

#### 知识点：食物链和食物网

**第一步：掌握食物链（20分钟）**

**食物链的定义：**
生产者和消费者之间通过吃与被吃形成的链状关系。

**食物链的书写规则（重要！）：**

**格式：** 生产者 → 初级消费者 → 次级消费者 → 三级消费者

**举例：**
```
草 → 蚱蜢 → 青蛙 → 蛇 → 鹰
```

**详细说明：**
- **草**：生产者（起点）
- **蚱蜢**：初级消费者（吃草）
- **青蛙**：次级消费者（吃蚱蜢）
- **蛇**：三级消费者（吃青蛙）
- **鹰**：四级消费者（吃蛇）

---

**书写注意事项（必须记住）：**

**✅ 正确做法：**
1. 箭头方向：从被吃的指向吃的（能量流动方向）
2. 起点必须是生产者（绿色植物）
3. 不包括分解者
4. 不包括非生物成分

**❌ 常见错误：**
1. 箭头方向反了：蛇 ← 青蛙（错误！）
2. 起点不是植物：蚱蜢 → 青蛙（错误！缺少生产者）
3. 包括分解者：草 → 兔子 → 细菌（错误！）
4. 包括非生物：阳光 → 草（错误！）

---

**例题练习：**

**判断下列食物链是否正确：**

1. 草 → 兔子 → 狼 （  ）
2. 阳光 → 草 → 兔子 （  ）
3. 狼 → 兔子 → 草 （  ）
4. 草 → 兔子 → 狼 → 细菌 （  ）
5. 草 → 蚱蜢 → 青蛙 → 蛇 （  ）

**答案：** ✅ ❌ ❌ ❌ ✅

---

**第二步：理解食物网（10分钟）**

**食物网的定义：**
在一个生态系统中，多条食物链彼此交错连接形成的网状结构。

**举例：**

```
        草
       /  \
      /    \
   蚱蜢    兔子
     |      |  \
   青蛙    狐狸  \
     |      |     |
    蛇  ----+-----+
     |
    鹰
```

**从这个食物网中可以找出的食物链：**
1. 草 → 蚱蜢 → 青蛙 → 蛇 → 鹰
2. 草 → 兔子 → 狐狸 → 蛇 → 鹰
3. 草 → 兔子 → 蛇 → 鹰

**技巧：** 数食物链条数，从每个生产者开始，沿着箭头走到底！

---

**第三步：能量流动（20分钟）**

**能量流动的特点（必须记住）：**

**1. 单向流动**
- 只能从生产者 → 消费者，不可逆
- 不能循环

**2. 逐级递减**
- 约10%传递给下一营养级
- 大部分能量用于自身生命活动或散失

---

**能量流动示意图：**

```
  阳光（100%）
     ↓
生产者（绿色植物）
  （约1%被固定）
     ↓（约10%传递）
初级消费者（植食性动物）
     ↓（约10%传递）
次级消费者（肉食性动物）
     ↓（约10%传递）
三级消费者
```

**能量去向：**
- 约10%：传递给下一营养级
- 约90%：
  - 自身呼吸消耗
  - 未被利用（排泄物、遗体）
  - 散失到环境中

---

**重要结论：**

1. **营养级越高，获得能量越少**
   - 所以高级消费者数量较少

2. **食物链一般不超过4-5个营养级**
   - 因为能量逐级递减，传递到顶端太少

3. **生态系统中植物数量最多**
   - 因为它们是能量的来源

---

**第四步：物质循环（10分钟）**

**物质循环的特点：** 循环流动（与能量流动不同）

**循环过程：**

```
    无机物（CO₂、水、无机盐）
           ↓
    生产者（光合作用）
           ↓
        有机物
           ↓
        消费者
           ↓
    遗体、排泄物
           ↓
        分解者
           ↓
        无机物
           ↓
    （重新被植物利用）
```

**对比：能量流动 vs 物质循环**

| 项目 | 能量流动 | 物质循环 |
|------|---------|---------|
| 方向 | 单向流动 | 循环流动 |
| 特点 | 逐级递减 | 反复利用 |
| 来源 | 太阳能 | 环境中的物质 |
| 去向 | 散失到环境 | 回到环境 |
