#### 📖 技巧一：角的拼凑（30分钟）

**核心思想：** 当两个角不直接相等时，通过加减同一个角使其相等

**公式：**
```
若 ∠ACD = ∠BCE
则 ∠ACD + ∠DCB = ∠BCE + ∠DCB
即 ∠ACB = ∠DCE
```

**🔥 记忆口诀：** "两角相等加同角，结果仍然是相等"

---

**【经典例题1】** 如图，∠1 = ∠2，AC = BD，求证：△ACE ≌ △BDF

```
    A————C————B
     \   |   /
      \  |  /
       \ | /
        \|/
    E————F————D
```

其中∠1 = ∠ACE，∠2 = ∠BDF

**分析：**
- 要用SAS判定，需要：AC = BD（✓已知）、∠? = ∠?、CE = DF（?）
- ∠1 = ∠2 不是夹角，需要转化
- 观察：∠ACF = ∠1 + ∠1'，∠BDE = ∠2 + ∠2'

**证明：**
```
∵ ∠1 = ∠2 （已知）
∴ ∠1 + ∠ECF = ∠2 + ∠ECF
即 ∠ACF = ∠BDE

在△ACF和△BDE中
∵ AC = BD      （已知）
  ∠ACF = ∠BDE  （已证）
  CF = DE      （已知）
∴ △ACF ≌ △BDE （SAS）
```

**📌 技巧要点：** 
1. 发现两个小角相等
2. 加上同一个角，得到两个大角相等
3. 用大角参与证明

---

**【变式训练】** 如图，∠BAD = ∠CAE，AB = AC，AD = AE，求证：BD = CE

```
        A
       /|\
      / | \
     B——D——E——C
```

**提示：** ∠BAD + ∠DAC = ∠CAE + ∠DAC → ∠BAC = ∠DAE

---

#### 📖 技巧二：截长补短（30分钟）

**核心思想：** 证明线段关系时，在长线段上截取等于短线段的一段

**两种方法：**
1. **截长：** 在长边上截取等于短边的线段
2. **补短：** 将短边延长到等于长边

---

**【经典例题2·截长法】** 在△ABC中，AB > AC，D是AB上一点，AD = AC，求证关系

```
       A
      /|\
     / | \
    D——+——\
   /   |   \
  B————+————C
```

**证明：**
```
在AB上取点D，使AD = AC（截长）

∵ AD = AC
∴ △ADC是等腰三角形
∴ ∠ADC = ∠ACD （等边对等角）
```

---

**【经典例题3·补短法】** 已知AB < CD，求证某结论

**方法：** 延长AB到E，使AE = CD

```
A————B——E
```

---

#### 📖 技巧三：边的等量代换（25分钟）

**核心思想：** 用已知的等量关系替换证明中的边

**常见形式：**
```
若 AB = CD，BC = EF
要证 AB + BC = CD + EF
则用 CD 代换 AB，用 EF 代换 BC
```

---

**【经典例题4】** 如图，AB = CD，E是BC的中点，求证：AE = DE

```
  A————B————E————C————D
```

**证明：**
```
∵ E是BC的中点
∴ BE = CE

∵ AB = CD （已知）
∴ AB + BE = CD + CE （等量代换）
即 AE = DE
```

---

**【经典例题5】** 如图，D是BC的中点，AD ⊥ BC，求证：AB = AC

```
       A
      /|\
     / | \
    /  |  \
   B———D———C
```

**证明：**
```
∵ D是BC的中点 （隐藏条件→BD = CD）
∴ BD = CD

∵ AD ⊥ BC （隐藏条件→∠ADB = ∠ADC = 90°）
∴ ∠ADB = ∠ADC = 90°

在△ABD和△ACD中
∵ BD = CD      （已证）
  ∠ADB = ∠ADC  （已证）
  AD = AD      （公共边）
∴ △ABD ≌ △ACD （SAS）
∴ AB = AC     （全等三角形对应边相等）
```

### ⏰ 10:30-12:00 | 生物·遗传概率计算（90分钟）

#### 📖 知识点一：遗传概率基础公式

**概率计算公式：**
```
某基因型概率 = 该基因型个数 / 子代总数
```

**基本遗传比例：**

| 亲本组合 | 子代基因型比 | 子代表现型比 |
|---------|------------|------------|
| AA × AA | 全AA | 全显性 |
| AA × Aa | AA:Aa = 1:1 | 全显性 |
| AA × aa | 全Aa | 全显性 |
| Aa × Aa | AA:Aa:aa = 1:2:1 | 显:隐 = 3:1 |
| Aa × aa | Aa:aa = 1:1 | 显:隐 = 1:1 |
| aa × aa | 全aa | 全隐性 |

---

#### 📖 知识点二：概率计算方法

**方法一：棋盘法（Punnett Square）**

```
Aa × Aa 的遗传分析：

        ♂
       A    a
    ┌────┬────┐
  A │ AA │ Aa │ ♀
    ├────┼────┤
  a │ Aa │ aa │
    └────┴────┘

结果：AA:Aa:aa = 1:2:1
```

**方法二：分枝法**

```
Aa × Aa

♂配子：A (1/2) 和 a (1/2)
♀配子：A (1/2) 和 a (1/2)

AA = 1/2 × 1/2 = 1/4
Aa = 1/2 × 1/2 × 2 = 1/2
aa = 1/2 × 1/2 = 1/4
```

---

#### 📝 遗传概率例题

**【例题1】** 豌豆高茎（D）对矮茎（d）为显性，Dd × Dd 子代中：
1. 高茎豌豆的概率是多少？
2. 矮茎豌豆的概率是多少？
3. 纯合子的概率是多少？

**解答：**
```
Dd × Dd 子代：DD : Dd : dd = 1 : 2 : 1

1. 高茎概率 = DD + Dd = 1/4 + 2/4 = 3/4 = 75%

2. 矮茎概率 = dd = 1/4 = 25%

3. 纯合子概率 = DD + dd = 1/4 + 1/4 = 1/2 = 50%
```

---

**【例题2·条件概率】** Dd × Dd 的子代中，取一株高茎豌豆，该豌豆是杂合子的概率是多少？

**解答：**
```
高茎豌豆包括：DD(1份) + Dd(2份) = 3份

其中杂合子Dd = 2份

所以：P(杂合子|高茎) = 2/3 ≈ 66.7%
```

**📌 这是条件概率题！分母是"高茎豌豆"，不是全部子代！**

---

**【例题3·连续概率】**

某对夫妇均为Aa（卷舌基因），第一个孩子是aa（不能卷舌），问：
1. 第二个孩子是aa的概率？
2. 两个孩子都是aa的概率？

**解答：**
```
1. 每次生育是独立事件，P(aa) = 1/4

2. P(两个都是aa) = 1/4 × 1/4 = 1/16
```

#### 📖 知识点三：测交实验

**测交定义：** 让未知基因型个体与隐性纯合子杂交

**作用：** 检验未知个体的基因型

```
被测个体（?） × 隐性纯合子(aa)

情况1：如果被测是AA
  AA × aa → 全Aa（全为显性）

情况2：如果被测是Aa
  Aa × aa → Aa:aa = 1:1（显:隐 = 1:1）
```

**📌 测交口诀：** "测交测交一比一，说明亲本是杂合"
