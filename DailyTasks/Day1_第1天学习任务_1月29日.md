#### 一、基础知识梳理（15分钟）

**1. 有理数的分类**

```
        有理数
         |
    _____|_____
   |           |
  整数        分数
   |           |
正整数      正分数
   0        负分数
负整数
```

**2. 数轴三要素**
- 原点（0的位置）
- 正方向（通常向右）
- 单位长度

**3. 重要概念**
- **相反数**：只有符号不同的两个数（如3和-3）
- **倒数**：乘积为1的两个数（如2和1/2）
- **绝对值**：数轴上表示数的点到原点的距离
  - |a| = a（a≥0）
  - |a| = -a（a<0）

---

#### 二、有理数运算法则（25分钟）

**1. 加法法则**

| 情况 | 法则 | 例题 |
|------|------|------|
| 同号两数相加 | 取相同符号，绝对值相加 | (-3) + (-5) = -(3+5) = -8 |
| 异号两数相加 | 取绝对值较大数的符号，绝对值相减 | (-7) + 5 = -(7-5) = -2 |
| 互为相反数 | 和为0 | 3 + (-3) = 0 |
| 一个数与0相加 | 仍得这个数 | -5 + 0 = -5 |

**加法运算律：**
- 交换律：a + b = b + a
- 结合律：(a + b) + c = a + (b + c)

**2. 减法法则**

**核心：** 减去一个数，等于加上这个数的相反数

```
a - b = a + (-b)
```

**例题：**
- 5 - 8 = 5 + (-8) = -3
- (-3) - (-7) = -3 + 7 = 4
- 0 - 6 = 0 + (-6) = -6

**3. 乘法法则**

| 符号情况 | 结果 | 例题 |
|----------|------|------|
| 同号相乘 | 正数 | (-3) × (-4) = 12 |
| 异号相乘 | 负数 | (-5) × 6 = -30 |
| 任何数×0 | 0 | 7 × 0 = 0 |

**乘法规律：**
- 几个不为0的数相乘，负因数的个数为偶数时，积为正数
- 几个不为0的数相乘，负因数的个数为奇数时，积为负数

**乘法运算律：**
- 交换律：ab = ba
- 结合律：(ab)c = a(bc)
- 分配律：a(b+c) = ab + ac

**4. 除法法则**

**核心：** 除以一个数（不为0），等于乘以这个数的倒数

```
a ÷ b = a × (1/b)  (b≠0)
```

**例题：**
- 12 ÷ (-3) = 12 × (-1/3) = -4
- (-15) ÷ (-5) = -15 × (-1/5) = 3

**5. 乘方运算**

**定义：** a^n = a × a × ... × a（n个a相乘）

**符号规律：**
- 正数的任何次幂都是正数
- 负数的奇数次幂是负数
- 负数的偶数次幂是正数

**例题：**
- (-2)³ = (-2) × (-2) × (-2) = -8
- (-2)⁴ = (-2) × (-2) × (-2) × (-2) = 16
- -2³ = -(2 × 2 × 2) = -8（注意：底数是2，不是-2）

---

#### 三、混合运算顺序（10分钟）

**运算顺序：**
1. 先算乘方
2. 再算乘除
3. 最后算加减
4. 有括号先算括号里的

**例题：**
```
计算：(-2)² + 3 × (-4) - 8 ÷ (-2)

步骤：
① 先算乘方：(-2)² = 4
② 再算乘除：3 × (-4) = -12，8 ÷ (-2) = -4
③ 最后算加减：4 + (-12) - (-4) = 4 - 12 + 4 = -4
```

---

#### 四、例题精讲（10分钟）

**例1：** 计算：(-3) + 5 + (-7) + 2

**解析：**
方法一（按顺序）：(-3) + 5 = 2，2 + (-7) = -5，-5 + 2 = -3

方法二（交换律）：[(-3) + (-7)] + (5 + 2) = -10 + 7 = -3

**例2：** 计算：5 - 8 + 3 - (-2)

**解析：**
先把减法变加法：5 + (-8) + 3 + 2
= [5 + 3 + 2] + (-8)
= 10 - 8
= 2

**例3：** 计算：(-2) × 3 × (-4) × (-5)

**解析：**
负数有3个（奇数），所以结果为负数
|-2 × 3 × 4 × 5| = 120
所以原式 = -120

**例4：** 计算：(-1)^2023

**解析：**
2023是奇数，负数的奇数次幂是负数
所以 (-1)^2023 = -1

#### 一、精读教材（30分钟）

**重点内容一：运动系统的组成**

```
运动系统 = 骨 + 关节 + 骨骼肌

骨：杠杆作用
关节：支点作用
骨骼肌：动力作用
```

**重点内容二：关节的结构**

```
关节的结构（由外到内）：
│
├─ 关节囊（最外层）：保护关节
│   ├─ 外层：结缔组织
│   └─ 内层：分泌滑液
│
├─ 关节腔：含滑液，减少摩擦
│
└─ 关节面：由关节头和关节窝组成
    └─ 关节软骨：覆盖在关节面上，减少摩擦，缓冲震动
```

**记忆口诀：**
"囊腔头窝加软骨，保护润滑是功能"

**重点内容三：骨骼肌的特性**

- **受刺激能收缩**：骨骼肌受神经传来的刺激时收缩
- **收缩牵动骨**：骨骼肌收缩时，牵动骨绕关节活动，产生运动
- **成对协作**：每块骨骼肌都不能单独完成一个动作，需要多块骨骼肌协同配合

**例子：屈肘和伸肘**
- 屈肘：肱二头肌收缩，肱三头肌舒张
- 伸肘：肱二头肌舒张，肱三头肌收缩

**重点内容四：动物的行为**

```
动物的行为
│
├─ 先天性行为（本能）
│   ├─ 特点：生来就有，不学就会
│   ├─ 例子：蜘蛛织网、鸟类筑巢、蜜蜂采蜜
│   └─ 由遗传物质决定
│
└─ 学习行为（后天获得）
    ├─ 特点：出生后通过学习获得
    ├─ 例子：狗做算术、鹦鹉学舌、海豚表演
    └─ 建立在先天性行为基础上
```

**区分方法：**
- 先天性行为：生来就有，不需要学习
- 学习行为：需要在生活过程中学习和练习才能获得

#### 一、精读教材（35分钟）

**课题1：我与社会**

**核心观点1：个人离不开社会**

**原因：**
1. **人的生存离不开社会**
   - 我们吃的粮食、穿的衣服、住的房子，都是社会提供的
   - 一个人无法生产生活所需的一切

2. **人的成长离不开社会**
   - 我们在学校学习知识
   - 在社会中学习生活技能
   - 社会为我们提供成长的环境

3. **人的价值实现离不开社会**
   - 个人的才能需要在社会中发挥
   - 个人的价值需要社会认可
   - 离开社会，个人无法实现自身价值

**核心观点2：社会离不开个人**

**原因：**
1. **社会是由一个个具体的人组成的**
   - 没有个人，就没有社会

2. **社会的发展需要每个人的贡献**
   - 每个人都是社会的一分子
   - 每个人都在为社会做贡献

**核心观点3：个人与社会相互依存**

```
个人 ←─────→ 社会
     相互依存
     相互影响
```

---

**课题2：网络改变世界**

**核心观点1：网络的积极影响**

1. **拓宽了交往范围**
   - 可以认识世界各地的朋友
   - 打破了时空限制

2. **提供了学习平台**
   - 网络课程
   - 在线图书馆
   - 学习软件

3. **方便了日常生活**
   - 网上购物
   - 在线支付
   - 信息查询

4. **促进了民主政治**
   - 网络问政
   - 网络监督
   - 表达意见

**核心观点2：网络的消极影响**

1. **信息泛滥，真假难辨**
   - 虚假信息
   - 网络谣言
   - 诈骗信息

2. **网络暴力**
   - 语言攻击
   - 人肉搜索
   - 网络欺凌

3. **沉迷网络**
   - 影响学习
   - 影响健康
   - 影响人际交往

4. **侵犯隐私**
   - 个人信息泄露
   - 隐私被公开

**核心观点3：正确使用网络**

**对国家：**
- 加强网络管理
- 完善网络法律
- 打击网络犯罪

**对个人：**
1. **遵守网络道德**
   - 不传播虚假信息
   - 不进行网络暴力
   - 尊重他人隐私

2. **合理安排时间**
   - 不沉迷网络
   - 控制上网时间
   - 先学习后娱乐

3. **提高辨别能力**
   - 不轻信网络信息
   - 不随意透露个人信息
   - 不参与网络赌博

4. **自我保护**
   - 不随意添加陌生人
   - 不随意点击链接
   - 不随意扫描二维码
   - 保护账号密码

☐ 粗心大意
☐ 审题错误

【知识点】
有理数乘法：负数个数决定积的符号
- 负数个数为偶数 → 积为正数
- 负数个数为奇数 → 积为负数

【解题步骤】
Step 1: 数负数个数：-2, -4, -1，共3个（奇数）
Step 2: 计算绝对值：2 × 3 × 4 × 1 = 24
Step 3: 确定符号：负数个数为奇数，所以结果为负
Step 4: 最终答案：-24

【易错点提醒】
⚠️ 注意区分负数个数的奇偶性
⚠️ 先数清楚有几个负数，再计算

3) (-1) × (-2) × (-3) × (-4) × (-5) = ?
