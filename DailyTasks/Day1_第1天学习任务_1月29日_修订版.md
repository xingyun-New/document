#### 知识点：分式的基本性质

**学习流程：**

**第一步：理解分式的定义（10分钟）**

**1. 什么是分式？**
- 定义：形如 A/B 的式子，其中A、B是整式，**B中含有字母**
- 注意：分母中必须含有字母才是分式

**例子：**
- (x+1)/(x-2) ← 是分式（分母含有字母x）
- 3/5 ← 不是分式（分母是常数）
- a²/(b+1) ← 是分式（分母含有字母b）

**2. 分式有意义的条件**
- 条件：分母 B ≠ 0
- 原因：分母不能为0

**例题：**
```
① 分式 (x-1)/(x+2) 有意义的条件是：x+2≠0，即 x≠-2

② 分式 1/(x²-4) 有意义的条件是：x²-4≠0，即 x≠±2
```

**3. 分式的值为0的条件**
- 条件：分子 A = 0 且 分母 B ≠ 0
- 记忆：分子为0，分母不为0

**例题：**
```
分式 (x²-1)/(x-1) 的值为0的条件是：
x²-1=0 且 x-1≠0
解得：x=-1（x=1不符合，因为分母为0）
```

---

**第二步：掌握分式的基本性质（25分钟）**

**核心性质（必须背诵！）：**

**性质1：** A/B = (A×C)/(B×C) （C≠0）
- 分子分母同乘一个不为0的数，分式值不变

**性质2：** A/B = (A÷C)/(B÷C) （C≠0）
- 分子分母同除以一个不为0的数，分式值不变

**记忆口诀：**
> 同乘同除值不变，分母不为零在前

**应用1：约分**
- 定义：利用分式基本性质，约去分子分母的公因式，化为最简分式
- 步骤：
  1. 分子分母分别因式分解
  2. 找出公因式
  3. 约去公因式

**例题1：数字系数约分**
```
(6x²y)/(9xy²) = (6xy·x)/(9xy·y) = 2x/3y

步骤：
①找公因式：6和9的最大公因数是3，x²和x的公因式是x，y和y²的公因式是y
②约去：3xy
③结果：2x/3y
```

**例题2：因式分解后约分**
```
(x²-4)/(x+2) = [(x+2)(x-2)]/(x+2) = x-2

步骤：
①分子因式分解：x²-4 = (x+2)(x-2)（平方差公式）
②约去公因式：x+2
③结果：x-2
```

**例题3：复杂约分**
```
(a²-4a+4)/(a²-4) = (a-2)²/[(a+2)(a-2)] = (a-2)/(a+2)

步骤：
①分子因式分解：a²-4a+4 = (a-2)²（完全平方公式）
②分母因式分解：a²-4 = (a+2)(a-2)（平方差公式）
③约去公因式：a-2
④结果：(a-2)/(a+2)
```

**常用因式分解公式（必须记住！）：**
- 平方差：a²-b² = (a+b)(a-b)
- 完全平方：a²±2ab+b² = (a±b)²
- 提公因式：ab+ac = a(b+c)

#### 知识点：遗传学基础

**学习流程：**

**第一步：理解三大核心概念（30分钟）**

**概念1：DNA、基因、染色体的关系（10分钟）**

**关系图：**
```
         细胞
           ↓
        细胞核
           ↓
        染色体 ← 能被碱性染料染成深色的物质
           ↓（由...组成）
   DNA + 蛋白质
           ↓（DNA包含）
         基因 ← DNA上有遗传效应的片段
           ↓（控制）
         性状 ← 生物的形态特征（如高矮、颜色）
```

**核心概念（必须背诵）：**
1. **染色体**：细胞核中能被碱性染料染成深色的物质
2. **DNA**：脱氧核糖核酸，是主要的遗传物质
3. **基因**：DNA上有遗传效应的片段，控制性状
4. **关系**：
   - 1条染色体 = 1个DNA分子 + 蛋白质
   - 1个DNA分子含有多个基因
   - 基因控制性状

**记忆口诀：**
> 染色体含DNA，DNA片段是基因，基因控制生物状

**例子：**
- 眼睛的颜色由基因控制
- 头发的卷直由基因控制
- 身高由基因控制

---

**概念2：基因型与表现型（10分钟）**

**基因型：** 生物体内的基因组成
- 用字母表示：AA、Aa、aa
- 大写字母（A）表示显性基因
- 小写字母（a）表示隐性基因

**表现型：** 生物体表现出来的性状
- 如：高茎、矮茎、红花、白花、双眼皮、单眼皮

**对应关系表：**

| 基因型 | 表现型 | 说明 |
|-------|--------|------|
| AA | 显性性状 | 显性纯合子 |
| Aa | 显性性状 | 杂合子（显性基因A表现） |
| aa | 隐性性状 | 隐性纯合子（隐性基因成对才表现） |

**例子：豌豆的茎高**
- 基因型 DD → 表现型：高茎
- 基因型 Dd → 表现型：高茎（D是显性，只要有就表现）
- 基因型 dd → 表现型：矮茎（d是隐性，成对才表现）

**关键点：**
- ⚠️ AA和Aa的基因型不同，但表现型相同（都是显性）
- ⚠️ 只有aa才表现隐性性状

---

**概念3：显性性状与隐性性状（10分钟）**

**显性性状：** 在杂合子中表现出来的性状
- 控制显性性状的基因叫**显性基因**（用大写字母表示，如A、D、T）
- 只要有一个显性基因就能表现

**隐性性状：** 在杂合子中未表现出来的性状
- 控制隐性性状的基因叫**隐性基因**（用小写字母表示，如a、d、t）
- 必须成对（aa）才能表现

**判断方法：**
```
        杂合子Aa
         ↓表现
         A性状 ← 显性性状
         
         a性状 ← 隐性性状（未表现）
```

**常见例子：**

| 性状 | 显性 | 隐性 |
|------|------|------|
| 眼皮 | 双眼皮（A） | 单眼皮（a） |
| 耳垂 | 有耳垂（E） | 无耳垂（e） |
| 豌豆茎高 | 高茎（D） | 矮茎（d） |
| 豌豆种子 | 圆粒（R） | 皱粒（r） |

**记忆口诀：**
> 大写显性小写隐，显性一个就能现，隐性必须成对见
