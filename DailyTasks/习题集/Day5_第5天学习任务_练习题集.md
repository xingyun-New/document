# 📋 答案与解析

## 第一部分：数学答案

### 基础证明题答案（1-10题）

**第1题答案：**
```
证明：在△ABD和△ACD中
∵ AB = AC（已知）
  BD = CD（已知）
  AD = AD（公共边）
∴ △ABD ≌ △ACD（SSS）
```
**答题技巧：** 找公共边AD是关键！

---

**第2题答案：**
```
证明：在△AOB和△COD中
∵ OA = OC（已知）
  OB = OD（已知）
  ∠AOB = ∠COD（已知/对顶角相等）
∴ △AOB ≌ △COD（SAS）
```
**答题技巧：** SAS判定，注意角是两边的夹角。

---

**第3题答案：**
```
证明：在△ABC和△CDA中
∵ AB∥CD（已知）
∴ ∠BAC = ∠DCA（两直线平行，内错角相等）
又∵ AB = CD（已知）
  AC = CA（公共边）
∴ △ABC ≌ △CDA（SAS）
```
**答题技巧：** 利用平行线性质产生角相等！

---

**第4题答案：**
```
证明：在△ABC和△DEF中
∵ ∠A = ∠D（已知）
  ∠B = ∠E（已知）
  AB = DE（已知）
∴ △ABC ≌ △DEF（AAS）
```
**答题技巧：** 两角一边用AAS或ASA。

---

**第5题答案：**
```
证明：在△ACD和△BDC中
∵ AC = BD（已知）
  AD = BC（已知）
  CD = DC（公共边）
∴ △ACD ≌ △BDC（SSS）
```
**答题技巧：** 注意对应顶点的顺序！

---

**第6题答案：**
```
证明：在△ABD和△CBD中
∵ AB = CB（已知）
  ∠ABD = ∠CBD（已知）
  BD = BD（公共边）
∴ △ABD ≌ △CBD（SAS）
```
**答题技巧：** 角平分线问题常用SAS。

---

**第7题答案：**

∵ ∠A = ∠C = 90°（已知）
  AE = CF（已知）
  AB = CD（已知/需要证明或题目给出）
∴ Rt△ABE ≌ Rt△CDF（HL）
```
**答题技巧：** 直角三角形优先考虑HL判定。

---

**第8题答案：**

**第9题答案：**
```
证明：在△ABC和△EDC中
∵ ∠1 = ∠2（已知）→ ∠ACB = ∠ECD
