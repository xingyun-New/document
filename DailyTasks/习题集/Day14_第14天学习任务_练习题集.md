### 三、解答题答案

**40题答案：**
```
AB² = AC² + BC² = 64 + 36 = 100
AB = 10cm

用面积法：
S = ½·AC·BC = ½·AB·CD
½·8·6 = ½·10·CD
24 = 5·CD
CD = 4.8cm
```

**42题答案：**
```
(1) A和B纵坐标相同
AB = |4-(-2)| = 6

(2) B和C横坐标相同
BC = |3-(-1)| = 4

(3) S = ½·AB·BC = ½·6·4 = 12
```

**48题答案：**
```
AB = √(AC² + BC²) = √(9+16) = 5

面积法：
½·AC·BC = ½·AB·CD
½·3·4 = ½·5·CD
6 = 2.5·CD
CD = 2.4
```

**50题答案：**
```
(1) AB = √[(4-0)² + (0-3)²] = √(16+9) = 5
    BC = |4-(-2)| = 6
    AC = √[(-2-0)² + (0-3)²] = √(4+9) = √13

(2) AB² + AC² = 25 + 13 = 38 ≠ 36 = BC²
    不是直角三角形

    AB = AC? 5 ≠ √13
    不是等腰三角形

    △ABC是不等边三角形（斜三角形）
```
