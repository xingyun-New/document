### 一、行程问题（10题）

**【相遇问题】**

1. 甲乙两人相距120千米，同时相向而行。甲的速度是每小时30千米，乙的速度是每小时50千米。问几小时后两人相遇？

2. A、B两城相距480千米，一列快车从A城开出，速度为80千米/时；一列慢车从B城开出，速度为60千米/时。两车同时相向而行，几小时后相遇？

3. 甲乙两人分别从相距100米的两地同时出发相向而行。甲的速度是3米/秒，乙的速度是2米/秒。几秒后两人相遇？

**【追及问题】**

4. 甲在乙前面20千米，甲的速度是40千米/时，乙的速度是50千米/时。乙几小时能追上甲？

**【往返问题】**

# 📋 答案与解析

## 第一部分：数学答案

### 行程问题答案（1-10题）

**第1题答案：**
```
解：设x小时后两人相遇
根据题意：30x + 50x = 120
        80x = 120
        x = 1.5
答：1.5小时后两人相遇。
```
**答题技巧：** 相遇问题核心公式：(速度1 + 速度2) × 时间 = 总路程

---

**第2题答案：**
```
解：设x小时后相遇
根据题意：80x + 60x = 480
        140x = 480
        x = 480/140 ≈ 3.43
答：约3.43小时后相遇。
```

---

**第3题答案：**
```
解：设x秒后相遇
根据题意：3x + 2x = 100
        5x = 100
        x = 20
答：20秒后两人相遇。
```

---

**第4题答案：**
```
解：设x小时后乙追上甲
根据题意：50x - 40x = 20
        10x = 20
        x = 2
答：2小时后乙能追上甲。
```
**答题技巧：** 追及问题核心公式：(快速 - 慢速) × 时间 = 相差路程

---

**第5题答案：**
```
解：小明15分钟走的路程 = 5 × (15/60) = 1.25千米
设爸爸x小时能追上
15x - 5x = 1.25
10x = 1.25
x = 0.125小时 = 7.5分钟
答：爸爸7.5分钟后能追上小明。
```

---

**第6题答案：**
```
解：第一辆车2小时走的路程 = 60 × 2 = 120千米
设x小时后追上
80x - 60x = 120
20x = 120
x = 6
答：6小时后能追上。
```

---

**第7题答案：**
```
解：从家到学校的路程 = 60 × 20 = 1200米
设回家需要x分钟
100x = 1200
x = 12
答：回家需要12分钟。
```
**答题技巧：** 往返问题核心：去时路程 = 回来路程

---

**第8题答案：**
```
解：设船在静水中速度为v千米/时，水流速度为u千米/时
顺水速度 = v + u = 60/3 = 20千米/时
逆水速度 = v - u = 60/4 = 15千米/时
联立：v + u = 20
      v - u = 15
解得：v = 17.5千米/时，u = 2.5千米/时
```
