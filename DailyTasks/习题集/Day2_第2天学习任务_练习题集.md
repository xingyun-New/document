# ✅ Day 2 练习题详细答案与解题技巧

## 📐 数学部分答案（含解题步骤）

### 一、分式乘除混合运算答案（1-15题）

**1. 3b/2**
- 解析：(2a/3b) × (9b²/4a) = (2a × 9b²)/(3b × 4a) = 18ab²/12ab = 3b/2
- 技巧：先约分再计算更简单

**2. 1/(x²y)**
- 解析：(x/y²) × (y/x³) = xy/(x³y²) = 1/(x²y)
- 步骤：约去公因式 xy

**3. b/a**
- 解析：[(a-b)/a] × [b/(a-b)] = b/a
- 技巧：(a-b)约去

**4. x-1**
- 解析：[(x²-1)/x] × [x/(x+1)] = [(x+1)(x-1)/x] × [x/(x+1)] = x-1
- 步骤：先因式分解 x²-1 = (x+1)(x-1)

**5. a**
- 解析：[(a²+2ab+b²)/b] × [a/(a+b)] = [(a+b)²/b] × [a/(a+b)] = a(a+b)/b = a
- 技巧：完全平方式 a²+2ab+b² = (a+b)²

**6. (x-2)/(x-2) = 1**
- 解析：[(x-2)²/(x²-4)] × (x+2) = [(x-2)²/[(x+2)(x-2)]] × (x+2) = (x-2)/(x-2) = 1
- 注意：x≠±2

**7. 6n/m**
- 解析：(3m/2n) ÷ (m²/4n²) = (3m/2n) × (4n²/m²) = 12mn²/2nm² = 6n/m

**8. b²/a**
- 解析：(a/b) ÷ (a²/b³) = (a/b) × (b³/a²) = ab³/a²b = b²/a

**9. 1/[(x-1)(x+1)] 或 1/(x²-1)**
- 解析：[(x+1)/(x-1)] ÷ (x+1) = [(x+1)/(x-1)] × [1/(x+1)] = 1/(x-1)(x+1)

**10. a+3**
- 解析：[(a²-9)/(a+1)] ÷ [(a-3)/(a+1)] = [(a+3)(a-3)/(a+1)] × [(a+1)/(a-3)] = a+3

**11. y(x-1)**
- 解析：[(x²-2x+1)/y] ÷ [(x-1)/y²] = [(x-1)²/y] × [y²/(x-1)] = y(x-1)

**12. 1**
- 解析：(x/y) × (y/z) ÷ (x/z) = (x/y) × (y/z) × (z/x) = xyz/xyz = 1

**13. 1/(4a)**
- 解析：(2a/b) ÷ (4a²/b) × (a/2) = (2a/b) × (b/4a²) × (a/2) = 2ab/8a² = 1/(4a)

**14. x/(x-1)²**
- 解析：[(x-1)/x] ÷ [(x-1)²/x²] × (1/x) = [(x-1)/x] × [x²/(x-1)²] × (1/x) = x/(x-1)²

**15. 1/(a-b)**
- 解析：{[(a+b)/(a-b)] × [(a+b)(a-b)]} ÷ (a+b)² = [(a+b)²/(a-b)] ÷ (a+b)² = 1/(a-b)

### 二、分式加减与通分运算答案（16-30题）

**16. 3/x**
- 解析：1/x + 2/x = (1+2)/x = 3/x
- 技巧：同分母直接加

**17. 1**
- 解析：a/b - (a-b)/b = [a-(a-b)]/b = b/b = 1

**18. 2/(x-1)**
- 解析：(x+1)/(x-1) - (x-1)/(x-1) = [(x+1)-(x-1)]/(x-1) = 2/(x-1)

**19. (a+3)/(a+b)**
- 解析：3/(a+b) + a/(a+b) = (3+a)/(a+b)

**20. (x+y)/(xy)**
- 解析：1/x + 1/y = y/(xy) + x/(xy) = (x+y)/(xy)
- 技巧：通分，最简公分母是 xy

**21. (2b-3a)/(ab)**
- 解析：2/a - 3/b = 2b/(ab) - 3a/(ab) = (2b-3a)/(ab)

**22. (x²-x+1)/[x(x-1)]**
- 解析：x/(x-1) - 1/x = x²/[x(x-1)] - (x-1)/[x(x-1)] = (x²-x+1)/[x(x-1)]

**23. 2x/(x²-1)**
- 解析：1/(x+1) + 1/(x-1) = (x-1)/[(x+1)(x-1)] + (x+1)/[(x+1)(x-1)] = 2x/(x²-1)

**24. (a-b)/(a-b) = 1**
- 解析：a/(a-b) + b/(b-a) = a/(a-b) - b/(a-b) = (a-b)/(a-b) = 1
- 注意：b-a = -(a-b)

**25. (x+1)/(x²-1) 或 1/(x-1)**
- 解析：x/(x²-1) + 1/(x+1) = x/[(x+1)(x-1)] + (x-1)/[(x+1)(x-1)] = (x+x-1)/(x²-1) = (2x-1)/(x²-1)
- 或简化：1/(x-1)

**26. (2x-5)/(x²-4)**
- 解析：3/(x-2) - (x+1)/(x²-4) = 3(x+2)/(x²-4) - (x+1)/(x²-4) = (3x+6-x-1)/(x²-4) = (2x+5)/(x²-4)

**27. (bc+ac+ab)/(abc)**
- 解析：1/a + 1/b + 1/c = bc/(abc) + ac/(abc) + ab/(abc) = (bc+ac+ab)/(abc)

**28. 1/b**
- 解析：(a+b)/(ab) - 1/a = (a+b)/(ab) - b/(ab) = a/(ab) = 1/b

**29. [x(x+y)-y(x-y)]/[(x-y)(x+y)] = (x²+2xy-y²)/(x²-y²)**
- 解析：x/(x-y) - y/(x+y) = x(x+y)/[(x-y)(x+y)] - y(x-y)/[(x-y)(x+y)]

**30. 答案：1**
- 解析：
  - 原式 = [1/(x-1) - 1/(x+1)] ÷ [2/(x²-1)]
  - = [(x+1)-(x-1)]/[(x-1)(x+1)] × [(x²-1)/2]
  - = 2/(x²-1) × (x²-1)/2 = 1
  - 当 x=3 时，值为 1

### 三、遗传概率计算答案（46-60题）

**46. 1/4（或25%）**
- 解析：Aa×Aa → AA:Aa:aa = 1:2:1，aa占1/4

- 计算：Aa/(AA+Aa) = 2/(1+2) = 2/3

- 解析：AA×Aa → AA:Aa = 1:1，无隐性
- 易错点：注意亲本是AA不是Aa

- 结合概率：1/2 × 1/2 = 1/4
