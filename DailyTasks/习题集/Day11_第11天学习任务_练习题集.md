### 三、解答题答案

**25题答案：**
```
解：根据勾股定理
AB² = AC² + BC²
AB² = 6² + 8² = 36 + 64 = 100
AB = 10cm
```

**26题答案：**
```
解：先求AB
AB² = AC² + BC² = 36 + 64 = 100
AB = 10

用面积法：
S = ½·AC·BC = ½·AB·CD
½·6·8 = ½·10·CD
24 = 5·CD
CD = 4.8cm
```

**27题答案：**
```
证明：∵ ∠ACB = ∠ADB = 90°
∴ Rt△ACB 和 Rt△ADB

在 Rt△ACB 和 Rt△ADB 中
     ┌ AB = AB    （公共斜边）
     └ AC = AD    （已知直角边）
∴ Rt△ACB ≌ Rt△ADB  （HL）
∴ BC = BD
```

**33题答案：**
```
(1) AB² = AC² + BC² = 9 + 16 = 25
    AB = 5cm

(2) 用面积法：
    S = ½·AC·BC = ½·AB·CD
    ½·3·4 = ½·5·CD
    6 = 2.5·CD
    CD = 2.4cm
```
