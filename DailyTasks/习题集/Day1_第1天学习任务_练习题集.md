# ✅ Day 1 练习题详细答案与解题技巧

## 📐 数学部分答案（含解题步骤）

### 一、填空题答案（1-10题）

**1. x≠-2**
- 解析：分母 x+2≠0，所以 x≠-2
- 技巧：分式有意义 → 分母不为0

**2. x≠±1**
- 解析：分母 x²-1≠0，即 (x+1)(x-1)≠0，所以 x≠1 且 x≠-1
- 技巧：先因式分解，再求解

**3. x=-2**
- 解析：|x|-2=0 且 x-2≠0，解得 |x|=2，即 x=±2，但 x≠2，所以 x=-2
- 易错点：别忘了检验分母不为0的条件

**4. x=2**
- 解析：x²-4=0 且 x+2≠0，解得 x=±2，但 x≠-2，所以 x=2
- 技巧：分式值为0 = 分子为0 且 分母不为0

**5. x=1/2**
- 解析：分母 2x-1=0，解得 x=1/2
- 技巧：无意义 = 分母为0

**6. 2x/3**
- 解析：(6xy)/(9y) = (6y·x)/(9y·1) = (6/9)·(x/1) = 2x/3
- 步骤：约去公因式 3y

**7. -2a/(3b)**
- 解析：(-8a²b)/(12ab²) = (-8ab·a)/(12ab·b) = -2a/(3b)
- 步骤：约去公因式 4ab

**8. x+1**
- 解析：(x²-1)/(x-1) = [(x+1)(x-1)]/(x-1) = x+1
- 技巧：平方差公式 a²-b² = (a+b)(a-b)

**9. a-2**
- 解析：(a²-4)/(a+2) = [(a+2)(a-2)]/(a+2) = a-2
- 技巧：先因式分解，再约分

**10. x-4**
- 解析：(x²-4x)/x = x(x-4)/x = x-4
- 技巧：提公因式法

### 二、化简题答案（11-20题）

**11. 3m/(5n)**
- 解析：(15m²n)/(25mn²) = (15mn·m)/(25mn·n) = 3m/(5n)
- 步骤：找公因式 5mn，约去

**12. -2x/(3y)**
- 解析：(-12x³y²)/(18x²y³) = (-12x²y²·x)/(18x²y²·y) = -2x/(3y)
- 技巧：系数、字母分别处理

**13. x+4**
- 解析：(x²-16)/(x-4) = [(x+4)(x-4)]/(x-4) = x+4
- 公式：x²-16 = (x+4)(x-4)

**14. (a-3)/(a+3)**
- 解析：
  - 分子：a²-9 = (a+3)(a-3)
  - 分母：a²+6a+9 = (a+3)²
  - 约去 (a+3)：(a-3)/(a+3)
- 易错点：分母是完全平方式

**15. (x-3)/(x-1)**
- 解析：
  - 分子：x²-2x-3 = (x-3)(x+1)
  - 分母：x²-1 = (x+1)(x-1)
  - 约去 (x+1)：(x-3)/(x-1)
- 技巧：十字相乘法因式分解

**16. 2(a-2)/(a+2)**
- 解析：
  - 分子：2a²-8 = 2(a²-4) = 2(a+2)(a-2)
  - 分母：a²+4a+4 = (a+2)²
  - 约去 (a+2)：2(a-2)/(a+2)
- 技巧：先提公因式

**17. (x-2)/(x+2)**
- 解析：
  - 分子：x²-4x+4 = (x-2)²
  - 分母：x²-4 = (x+2)(x-2)
  - 约去 (x-2)：(x-2)/(x+2)
- 公式：完全平方式

**18. 3/(x+1)**
- 解析：
  - 分子：3x-3 = 3(x-1)
  - 分母：x²-1 = (x+1)(x-1)
  - 约去 (x-1)：3/(x+1)

**19. a(a+1)/(a-1)**
- 解析：
  - 分子：a³-a = a(a²-1) = a(a+1)(a-1)
  - 分母：a²-2a+1 = (a-1)²
  - 约去 (a-1)：a(a+1)/(a-1)

**20. (x+2)/(x-2)**
- 解析：
  - 分子：x²+4x+4 = (x+2)²
  - 分母：x²-4 = (x+2)(x-2)
  - 约去 (x+2)：(x+2)/(x-2)

### 三、提高题答案（21-30题）

**21. x≠-3**

- 注意：a 是常数，不影响分式有意义条件

**29. x(x+1)/(x-1)**

  - 分母：x²-2x+1 = (x-1)²
  - 约去 (x-1)：x(x+1)/(x-1)

**30. (x-4)/(x+4)**
