#### 知识点：全等三角形5种判定定理

**学习流程：**

**第一步：理解全等三角形的定义（5分钟）**

**全等三角形：** 能够完全重合的两个三角形叫做全等三角形。

**性质：**
- 对应边相等
- 对应角相等

**记法：** △ABC≌△DEF（读作"三角形ABC全等于三角形DEF"）

⚠️ **注意：** 书写全等时，对应顶点要写在对应位置！

---

**第二步：掌握5种判定定理（35分钟）**

**判定定理（必须背诵）：**

**1. SSS（边边边）**
- **定理：** 三条边分别相等的两个三角形全等
- **条件：** AB=DE, AC=DF, BC=EF
- **结论：** △ABC≌△DEF

**例子：**
```
已知：AB=DE=3cm, AC=DF=4cm, BC=EF=5cm
结论：△ABC≌△DEF (SSS)
```

---

**2. SAS（边角边）**
- **定理：** 两边和它们的夹角分别相等的两个三角形全等
- **条件：** AB=DE, ∠A=∠D, AC=DF
- **结论：** △ABC≌△DEF

⚠️ **注意：** 必须是两边的夹角！

**例子：**
```
已知：AB=DE=5cm, ∠A=∠D=60°, AC=DF=4cm
结论：△ABC≌△DEF (SAS)
```

---

**3. ASA（角边角）**
- **定理：** 两角和它们的夹边分别相等的两个三角形全等
- **条件：** ∠A=∠D, AB=DE, ∠B=∠E
- **结论：** △ABC≌△DEF

⚠️ **注意：** 必须是两角的夹边！

**例子：**
```
已知：∠A=∠D=50°, AB=DE=6cm, ∠B=∠E=70°
结论：△ABC≌△DEF (ASA)
```

---

**4. AAS（角角边）**
- **定理：** 两角和其中一角的对边分别相等的两个三角形全等
- **条件：** ∠A=∠D, ∠B=∠E, BC=EF
- **结论：** △ABC≌△DEF

**例子：**
```
已知：∠A=∠D=45°, ∠B=∠E=60°, BC=EF=7cm
结论：△ABC≌△DEF (AAS)
```

---

**5. HL（斜边直角边）**
- **定理：** 斜边和一条直角边分别相等的两个直角三角形全等
- **条件：** AB=DE, AC=DF, ∠C=∠F=90°
- **结论：** Rt△ABC≌Rt△DEF

⚠️ **注意：** 只适用于直角三角形！

**例子：**
```
已知：Rt△ABC和Rt△DEF中，∠C=∠F=90°, AB=DE=10cm, AC=DF=8cm
结论：Rt△ABC≌Rt△DEF (HL)
```

---

**记忆口诀：**
> 三边三角两特殊  
> SSS、SAS、ASA、AAS  
> 直角三角形特殊，HL也能证全等

**对比记忆表：**

| 判定 | 条件 | 注意事项 | 适用范围 |
|------|------|---------|---------|
| SSS | 三边 | 无 | 所有三角形 |
| SAS | 两边+夹角 | 必须是夹角 | 所有三角形 |
| ASA | 两角+夹边 | 必须是夹边 | 所有三角形 |
| AAS | 两角+对边 | 任意对边 | 所有三角形 |
| HL | 斜边+直角边 | 仅直角三角形 | 直角三角形 |

---

**第三步：掌握证明格式（15分钟）**

**证明格式模板（必须掌握）：**

```
证明：在△ABC和△DEF中
∵ AB = DE（已知）
  AC = DF（已知）
  ∠A = ∠D（已知）
∴ △ABC ≌ △DEF（SAS）
∴ BC = EF（全等三角形对应边相等）
```

**格式要点：**
1. 开头写"证明：在△ABC和△DEF中"
2. 用"∵"表示"因为"
3. 用"∴"表示"所以"
4. 写明判定理由（SSS/SAS等）
5. 写明对应关系（对应边/角相等）

---

**第四步：背诵与默写（5分钟）**

**必背内容：**
1. SSS：三边分别相等
2. SAS：两边和夹角相等
3. ASA：两角和夹边相等
4. AAS：两角和对边相等
5. HL：斜边和直角边相等（直角三角形）

**默写：** 在笔记本上默写5种判定定理

**神经系统组成图：**

```
        神经系统
           |

   |               |
中枢神经系统    周围神经系统
   |               |
___|___         ___|___

|

```

**一、中枢神经系统（必背）：**

**1. 大脑**
- **位置：** 颅腔内，脑的最大部分
- **结构：** 两个大脑半球，表面有沟和回
- **功能：**
  - 感觉中枢（接收视觉、听觉、触觉等信息）
  - 运动中枢（控制身体运动）
  - 语言中枢（说话、理解语言、书写、阅读）
  - 思维中枢（思考、记忆、判断）

**2. 小脑**
- **位置：** 大脑后下方
- **功能：**
  - 协调运动，使动作准确
  - 维持身体平衡

**例子：** 骑自行车、跳舞需要小脑协调

**3. 脑干**
- **位置：** 大脑和脊髓之间
- **功能：**
  - 呼吸中枢
  - 心跳中枢
  - 血压调节中枢
  - 基本生命活动中枢

⚠️ **脑干受损会危及生命！**

**4. 脊髓**
- **位置：** 椎管内
- **功能：**
  - 反射中枢（如膝跳反射）
  - 传导通路（传递信息）

---

**二、周围神经系统：**

- **脑神经：** 12对，连接脑和头面部器官
- **脊神经：** 31对，连接脊髓和躯干、四肢

**功能：** 传递中枢神经系统和身体各部分之间的信息

### ⏰ 15:15-16:15 | 生物专项训练（60分钟）

> 📝 **目标：** 完成25道神经调节题，正确率65%以上

#### 知识点：反射和反射弧

**第一步：理解反射的概念（15分钟）**

**反射的定义：**
人体通过神经系统，对外界或内部的各种刺激所发生的有规律的反应。

**反射的特点：**
1. 需要完整的反射弧
2. 有规律的反应
3. 不受意识控制（简单反射）

**反射的意义：**
- 适应环境变化
- 保护身体
- 维持生命活动

---

**第二步：掌握反射弧（20分钟）**

**反射弧的5个环节（必须背诵，能画图）：**

```
感受器 → 传入神经 → 神经中枢 → 传出神经 → 效应器
```

**详细说明：**

**1. 感受器**
- 定义：感受刺激的结构
- 位置：皮肤、眼、耳、鼻、舌等
- 功能：接收刺激，产生神经冲动

**2. 传入神经**
- 功能：把神经冲动传到神经中枢

**3. 神经中枢**
- 位置：脑或脊髓
- 功能：接收信息，产生指令

**4. 传出神经**
- 功能：把神经冲动从神经中枢传到效应器

**5. 效应器**
- 定义：产生反应的结构
- 类型：肌肉、腺体
- 功能：做出反应（肌肉收缩或腺体分泌）

---

**举例1：膝跳反射**

```
叩击膝盖下方
    ↓
感受器（膝盖下的感受器）
    ↓
传入神经（把信号传到脊髓）
    ↓
神经中枢（脊髓中的反射中枢）
    ↓
传出神经（把指令传到大腿肌肉）
    ↓
效应器（大腿肌肉收缩）
    ↓
小腿前踢
```

**举例2：缩手反射**

```
手碰到热物体
    ↓
感受器（皮肤中的痛觉感受器）
    ↓
传入神经
    ↓
神经中枢（脊髓）
    ↓
传出神经
    ↓
效应器（手臂肌肉）
    ↓
手缩回来
```

⚠️ **注意：** 缩手反射的神经中枢在脊髓，不在大脑！所以反应很快！

---

**第三步：区分反射类型（10分钟）**

**反射的类型：**

**1. 简单反射（非条件反射）**
- 特点：生来就有的，不需要学习
- 神经中枢：脊髓或脑干
- 例子：
  - 膝跳反射
  - 缩手反射
  - 眨眼反射
  - 排尿反射
  - 婴儿吸吮反射

**2. 复杂反射（条件反射）**
- 特点：后天学习获得的
- 神经中枢：大脑皮层
- 例子：
  - 望梅止渴
  - 谈虎色变
  - 听到铃声流口水（巴甫洛夫的狗）
  - 看到红灯停下来

**对比表：**

| 项目 | 简单反射 | 复杂反射 |
|------|---------|---------|
| 形成时间 | 生来就有 | 后天学习 |
| 神经中枢 | 脊髓或脑干 | 大脑皮层 |
| 是否需要学习 | 不需要 | 需要 |
| 例子 | 膝跳反射 | 望梅止渴 |

**主题2深化：法律的作用**

核心观点1：法律规范作用
- 告诉人们可以做什么
- 必须做什么
- 应当做什么
- 不能做什么

核心观点3：违法行为的分类
```
违法行为分类：
├─ 民事违法行为（一般违法）
│  例：欠债不还、损坏他人财物
│  后果：承担民事责任
│
├─ 行政违法行为（一般违法）
│  例：闯红灯、扰乱公共秩序
│  后果：行政处罚（警告、罚款、拘留等）
│
└─ 刑事违法行为（严重违法=犯罪）
   例：抢劫、故意伤害、盗窃（数额较大）
   后果：刑事处罚（有期徒刑、无期徒刑等）
```

**主题3深化：权利与义务**

核心观点1：公民的基本权利
- 人身自由权
- 受教育权
- 财产权
- 政治权利

核心观点2：公民的基本义务
- 遵守宪法和法律
- 维护国家安全
- 依法纳税
- 服兵役

核心观点3：权利与义务的关系
- 权利和义务相统一
- 没有无权利的义务，也没有无义务的权利
- 享受权利的同时必须履行义务

**主题4：依法维权**

核心观点1：维权方式
- 协商和解
- 人民调解

- 诉讼（打官司）
