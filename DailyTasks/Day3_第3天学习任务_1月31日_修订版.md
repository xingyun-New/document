### ⏰ 08:30-09:30 | 数学重点突破（60分钟）

#### 知识点：幂的运算5条法则

**必须背诵的5条法则：**

1. **同底数幂相乘**：a^m · a^n = a^(m+n)
   - 底数不变，指数相加

2. **幂的乘方**：(a^m)^n = a^(mn)
   - 底数不变，指数相乘

3. **积的乘方**：(ab)^n = a^n · b^n
   - 每个因式分别乘方

4. **同底数幂相除**：a^m ÷ a^n = a^(m-n) (a≠0)
   - 底数不变，指数相减

5. **零指数和负指数**：
   - a^0 = 1 (a≠0)
   - a^(-n) = 1/a^n (a≠0)

**记忆口诀：**
> 同底相乘指数加，乘方指数要相乘  
> 积的乘方分别乘，同底相除指数减  
> 零次方等于1，负指数取倒数

**例题练习（30分钟）：**

2. (x²)³ = x⁶
3. (2x)³ = 8x³

#### 知识点：整式乘除法+因式分解

**1. 整式乘除法（20分钟）**

**单项式乘单项式：**
- (3x²) · (2x³) = 6x⁵

**单项式乘多项式：**
- 2x(x+3) = 2x² + 6x

**多项式乘多项式：**
- (x+2)(x+3) = x² + 5x + 6

**2. 乘法公式（20分钟）**

**平方差公式：**
(a+b)(a-b) = a² - b²

**完全平方公式：**
(a+b)² = a² + 2ab + b²
(a-b)² = a² - 2ab + b²

**3. 因式分解（20分钟）**

**方法1：提公因式法**
- ax + ay = a(x+y)

**方法2：公式法**
- a² - b² = (a+b)(a-b)
- a² + 2ab + b² = (a+b)²

**例题：**
```
① 因式分解：x² - 4 = (x+2)(x-2)
② 因式分解：x² + 4x + 4 = (x+2)²
```

### ⏰ 11:00-12:00 | 生物重点突破（60分钟）

#### 知识点：遗传综合题解题方法

**方法1：逆推法（已知子代推亲代）**

**例题：**
一对夫妇生了一个单眼皮（aa）的孩子，推断父母的基因型。

**解答：**
- 子代aa，说明父母各提供一个a
- 父母可能是：Aa × Aa、Aa × aa、aa × aa
- 如果父母都是双眼皮，则父母基因型为Aa × Aa

**方法2：分离比法（判断显隐性）**

**例题：**
豌豆高茎×高茎 → 子代高茎:矮茎 = 3:1

**推断：**
- 3:1是Aa × Aa的分离比
- 高茎是显性，矮茎是隐性

**方法3：概率计算法**

**例题：**
父母都是Aa，求：
- 第一个孩子是AA的概率：1/4
- 连续两个孩子都是AA的概率：1/4 × 1/4 = 1/16
- 至少有一个孩子是aa的概率：1 - (3/4 × 3/4) = 7/16

**学习流程：**

**第一步：精读教材（25分钟）**

**主题1：社会秩序与规则**

核心观点1：社会秩序的重要性
- 社会秩序是社会生活的一种有序化状态
- 包括：社会管理秩序、生产秩序、交通秩序、公共场所秩序等
- 社会正常运转需要秩序

核心观点2：社会规则维护秩序
- 规则是什么：人们共同遵守的办事规程和行为准则
- 规则的作用：
  ①调节人们的行为
  ②维护社会秩序
  ③保障社会运转

核心观点3：社会规则的种类
- 道德：依靠舆论、习惯、信念等维持
- 法律：依靠国家强制力保证实施
- 纪律：由组织制定，成员共同遵守

**主题2：法律的特征**

核心观点1：法律的三个基本特征（必背！）
```
特征一：法律是由国家制定或认可的
        （体现国家意志）
        
特征二：法律是由国家强制力保证实施的
        （最主要的特征，区别于其他规则）
        
特征三：法律对全体社会成员具有普遍约束力
        （法律面前人人平等）
```

核心观点2：法律与道德的关系

| 对比项 | 法律 | 道德 |
|-------|------|------|
| 产生方式 | 国家制定或认可 | 自然形成 |
| 实施手段 | 国家强制力 | 舆论、信念、习惯 |
| 约束范围 | 全体社会成员 | 自愿接受者 |
| 联系 | 相辅相成，法律以道德为基础，道德需法律支撑 |

**主题3：遵守规则**

核心观点1：为什么要遵守规则
- 规则保证社会秩序
- 规则保护我们的权利
- 违反规则要承担后果

核心观点2：如何遵守社会规则
- 发自内心地敬畏规则
- 将规则内化于心、外化于行
- 从小事做起，养成好习惯
- 自觉接受监督

核心观点3：违反规则的后果
- 违反道德：受到舆论谴责、良心不安
- 违反纪律：受到纪律处分
- 违反法律：承担法律责任（民事/行政/刑事）
