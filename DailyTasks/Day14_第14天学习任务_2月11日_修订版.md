### ⏰ 08:30-10:30 | 数学·函数初步（120分钟）

#### 📖 知识点一：函数的概念

**定义：** 一般地，在一个变化过程中，如果有两个变量x和y，并且对于x的每一个确定的值，y都有唯一确定的值与之对应，那么我们就说y是x的函数。

**表示：** y = f(x)

**关键词：**
- x：自变量
- y：因变量（函数值）
- 对应关系：函数

**🔥 判断是否为函数的关键：**
给定一个x值，y值是否**唯一确定**

---

#### 📖 知识点二：函数的表示方法

| 方法 | 示例 | 特点 |
|------|------|------|
| 解析法（公式法） | y = 2x + 1 | 精确，便于计算 |
| 列表法 | x和y的对应表格 | 直观，适合离散数据 |
| 图像法 | 坐标系中的图像 | 形象，易于观察趋势 |

---

#### 📖 知识点三：自变量的取值范围

**求自变量取值范围的原则：**

1. **分式：** 分母 ≠ 0
   - y = 1/(x-2) → x ≠ 2

2. **根式（偶次根）：** 被开方数 ≥ 0
   - y = √(x-3) → x ≥ 3

3. **实际问题：** 根据实际意义确定
   - 长度、时间等必须 > 0

---

#### 📝 函数例题

**【例题1】** 下列关系式中，y是x的函数的是（  ）
A. y² = x
B. y = 2x + 1
C. x + y = 0
D. y = ±√x

```
解析：
A. y² = x → y = ±√x，一个x对应两个y，不是函数 ✗
B. y = 2x + 1，一个x对应一个y，是函数 ✓
C. x + y = 0 → y = -x，是函数 ✓
D. y = ±√x，一个x对应两个y，不是函数 ✗

答案：B、C
```

**【例题2】** 函数 y = 1/(x-1) 的自变量取值范围是____

```
解：分母不能为0
    x - 1 ≠ 0
    x ≠ 1
    
答：x ≠ 1 或 x的取值范围是x ≠ 1
```

**【例题3】** 函数 y = √(2x-4) 的自变量取值范围是____

```
解：根号内必须 ≥ 0
    2x - 4 ≥ 0
    2x ≥ 4
    x ≥ 2
    
答：x ≥ 2
```

---

#### 📖 知识点四：函数值的计算

**方法：** 将x的值代入函数表达式

**【例题4】** 已知y = 3x - 2，求当x = 2时y的值

```
解：y = 3×2 - 2 = 6 - 2 = 4
```

**【例题5】** 已知f(x) = x² - 2x + 1，求f(3)

```
解：f(3) = 3² - 2×3 + 1 = 9 - 6 + 1 = 4
```
