### ⏰ 08:30-09:30 | 数学重点突破（60分钟）

#### 知识点：分式的乘除法

**核心公式（必须背诵）：**

**乘法法则：**
(a/b) × (c/d) = (a×c)/(b×d) = (ac)/(bd)

**除法法则：**
(a/b) ÷ (c/d) = (a/b) × (d/c) = (ad)/(bc)

**记忆口诀：**
> 乘法分子乘分子，分母乘分母  
> 除法等于乘倒数，除以谁就把谁倒

**学习内容（60分钟）：**

**1. 分式乘法（20分钟）**

**例题1：数字分式相乘**
```
2/3 × 5/7 = (2×5)/(3×7) = 10/21
```

**例题2：含字母的分式相乘**
```
x/y × a/b = (xa)/(yb)

(2x)/(3y) × (5y)/(4x) = (2x×5y)/(3y×4x) = (10xy)/(12xy) = 5/6
```

**例题3：先约分再相乘**
```
(x²-1)/(x+2) × (x+2)/(x-1) = [(x+1)(x-1)]/(x+2) × (x+2)/(x-1)
= [(x+1)(x-1)(x+2)]/[(x+2)(x-1)] = x+1
```

**2. 分式除法（20分钟）**

**关键：除以一个分式等于乘以它的倒数**

**例题1：**
```
3/4 ÷ 5/6 = 3/4 × 6/5 = 18/20 = 9/10
```

**例题2：**
```
a/b ÷ c/d = a/b × d/c = (ad)/(bc)
```

**例题3：**
```
(x²-4)/(x+1) ÷ (x-2)/(x+1) = (x²-4)/(x+1) × (x+1)/(x-2)
= [(x+2)(x-2)]/(x+1) × (x+1)/(x-2) = x+2
```

**3. 乘除混合运算（20分钟）**

**运算顺序：从左往右依次计算**

**例题：**
```
(x+1)/(x-1) × (x²-1)/(x+2) ÷ (x+1)/(x+2)

= (x+1)/(x-1) × [(x+1)(x-1)]/(x+2) × (x+2)/(x+1)

= [(x+1)(x+1)(x-1)(x+2)]/[(x-1)(x+2)(x+1)]

= x+1
```

**做5道练习题巩固**

### ⏰ 11:00-12:00 | 生物重点突破（60分钟）

#### 知识点：遗传图解的规范画法

**遗传图解标准格式（必须掌握）：**

```
P（亲代）：   Aa      ×      Aa
           (高茎)         (高茎)
             ↓
        生殖细胞：  A、a      A、a
             ↓         ⊗（受精）
F1（子代）：   AA      Aa      Aa      aa
          (高茎)   (高茎)  (高茎)  (矮茎)
         比例：   1   :   2   :   1
         
基因型比例：AA : Aa : aa = 1 : 2 : 1
表现型比例：高茎 : 矮茎 = 3 : 1
```

**关键符号：**
- **P**：亲代（Parents）
- **F1**：子一代（First Filial generation）
- **F2**：子二代
- **×**：杂交符号
- **⊗**：受精符号
- **( )**：表现型用括号

**学习内容（60分钟）：**

**1. 遗传规律（20分钟）**

**规律1：Aa × Aa**
- 子代基因型：AA : Aa : aa = 1 : 2 : 1
- 子代表现型：显性 : 隐性 = 3 : 1

**规律2：AA × Aa**
- 子代基因型：AA : Aa = 1 : 1
- 子代表现型：全部显性

**规律3：Aa × aa**
- 子代基因型：Aa : aa = 1 : 1
- 子代表现型：显性 : 隐性 = 1 : 1

**规律4：AA × aa**
- 子代基因型：全部 Aa
- 子代表现型：全部显性

**2. 画遗传图解（20分钟）**

练习画5个遗传图解：
1. Aa × Aa
2. AA × Aa
3. Aa × aa
4. AA × aa
5. aa × aa

**3. 遗传概率计算（20分钟）**

**例题：**
一对夫妇都是双眼皮（Aa），他们的孩子：
- 是双眼皮的概率：3/4
- 是单眼皮的概率：1/4
- 基因型为AA的概率：1/4
- 基因型为Aa的概率：1/2
