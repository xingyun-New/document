#### 一、单项式和多项式（15分钟）

**1. 单项式**

**定义：** 数字与字母的乘积

**例子：**
- 5x
- -3xy²
- 2πr
- a
- -7

**单项式的系数和次数：**

| 单项式 | 系数 | 次数 |
|--------|------|------|
| 5x | 5 | 1 |
| -3x²y | -3 | 3（2+1） |
| xy | 1 | 2（1+1） |
| -a²b³ | -1 | 5（2+3） |
| 6 | 6 | 0 |

**注意：**
- 系数包括符号
- 次数是所有字母指数的和
- 单独的数字也是单项式，次数为0

**2. 多项式**

**定义：** 几个单项式的和

**例子：**
- 3x + 2y
- x² - 2x + 1
- 5a²b - 3ab + 7

**多项式的项数和次数：**

| 多项式 | 项数 | 各项 | 次数 |
|--------|------|------|------|
| 3x + 2y | 2 | 3x, 2y | 1（最高次项的次数） |
| x² - 2x + 1 | 3 | x², -2x, 1 | 2 |
| 5a²b - 3ab | 2 | 5a²b, -3ab | 3 |

**注意：**
- 多项式的次数是最高次项的次数
- 常数项也算一项

---

#### 二、同类项（10分钟）

**定义：** 所含字母相同，并且相同字母的指数也相同的项

**判断方法：**

| 一组项 | 是否同类项 | 原因 |
|--------|-----------|------|
| 2x 和 3x | ✓ 是 | 字母相同，指数相同 |
| 2x² 和 3x² | ✓ 是 | 字母相同，指数相同 |
| 2xy 和 3yx | ✓ 是 | 字母相同（顺序无关），指数相同 |
| 2x 和 2x² | ✗ 不是 | 指数不同 |
| 2xy 和 2x | ✗ 不是 | 字母不同 |
| 2xy 和 3xz | ✗ 不是 | 字母不同 |
| 5 和 7 | ✓ 是 | 都是常数项 |

**关键点：**
1. 与系数无关（2x和5x是同类项）
2. 与字母顺序无关（xy和yx是同类项）
3. 字母和指数都要相同

---

#### 三、合并同类项（15分钟）

**法则：** 系数相加，字母和字母的指数不变

**例题：**

**例1：** 3x + 2x = ?
- 系数相加：3 + 2 = 5
- 字母不变：x
- 答案：5x

**例2：** 5a - 2a = ?
- 系数相加：5 + (-2) = 3
- 字母不变：a
- 答案：3a

**例3：** 2x² + 3x + x² - x = ?
- 先找同类项：2x²和x²，3x和-x
- 分别合并：2x² + x² = 3x²，3x - x = 2x
- 答案：3x² + 2x

**例4：** 3xy + 2x - xy + 5x = ?
- 同类项：3xy和-xy，2x和5x
- 合并：3xy - xy = 2xy，2x + 5x = 7x
- 答案：2xy + 7x

---

#### 四、去括号法则（15分钟）

**法则：**
1. 括号前是"+"，去括号后各项符号不变
2. 括号前是"-"，去括号后各项符号改变

**例题：**

**例1：** a + (b - c) = ?
- 括号前是"+"，符号不变
- 答案：a + b - c

**例2：** a - (b - c) = ?
- 括号前是"-"，符号改变
- 答案：a - b + c

**例3：** 2(x + y) = ?
- 分配律：2×x + 2×y
- 答案：2x + 2y

**例4：** -3(2x - y) = ?
- 分配律：-3×2x + (-3)×(-y)
- 答案：-6x + 3y

---

#### 五、整式加减综合（5分钟）

**步骤：**
1. 去括号
2. 合并同类项

**例题：** (3x + 2y) - (x - y)

**解：**
- 去括号：3x + 2y - x + y
- 合并同类项：(3x - x) + (2y + y) = 2x + 3y

#### 一、太平天国运动（1851-1864）（25分钟）

**时间轴：**

```
1851年 金田起义
  ↓
1853年 定都天京（南京）
  ↓
1856年 天京事变（内讧开始）
  ↓
1864年 天京陷落，运动失败
```

**详细内容：**

**1. 背景**
- 清政府腐败
- 阶级矛盾尖锐
- 外国侵略

**2. 领导人**
- 洪秀全（创立拜上帝教）

**3. 开始**
- 1851年，金田起义

**4. 建立政权**
- 1853年，攻占南京，改名天京，建立太平天国

**5. 纲领**
- 《天朝田亩制度》
  - 主张：废除封建土地所有制
  - 目标：平均分配土地
  - 评价：革命性但不能实现

**6. 失败**
- 1856年，天京事变，内部分裂
- 1864年，天京陷落，运动失败

**7. 意义**
- 沉重打击了中外反动势力
- 是中国近代史上规模最大的农民运动

**记忆口诀：**
"五一金田起义，五三定都天京，五六内讧开始，六四运动失败"
