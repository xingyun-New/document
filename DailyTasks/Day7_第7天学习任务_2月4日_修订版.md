### ⏰ 08:00-09:00 | 数学复习1（60分钟）

#### 复习内容：分式运算 + 整式运算

**一、分式运算（30分钟）**

**必须掌握的知识点：**

**1. 分式的基本概念**
- 分式定义：A/B，B中含字母
- 有意义条件：B ≠ 0
- 值为0条件：A = 0 且 B ≠ 0

**2. 分式的基本性质**
- A/B = (A×C)/(B×C) （C≠0）
- A/B = (A÷C)/(B÷C) （C≠0）

**3. 分式运算**
- **乘法：** (a/b) × (c/d) = (ac)/(bd)
- **除法：** (a/b) ÷ (c/d) = (a/b) × (d/c) = (ad)/(bc)
- **加减法：** 
  - 同分母：(a/c) ± (b/c) = (a±b)/c
  - 异分母：先通分，再加减

**4. 约分**
- 步骤：因式分解 → 找公因式 → 约去

**5. 常用公式**
- 平方差：a² - b² = (a+b)(a-b)
- 完全平方：(a±b)² = a² ± 2ab + b²

**一、全等三角形（30分钟）**

**必须掌握的知识点：**

**1. 全等三角形5种判定定理（必须背诵）**
- **SSS**：三边分别相等
- **SAS**：两边和夹角相等
- **ASA**：两角和夹边相等
- **AAS**：两角和对边相等
- **HL**：斜边和直角边相等（直角三角形）

**2. 证明格式**
```
证明：在△ABC和△DEF中
∵ AB = DE（已知）
  AC = DF（已知）
  ∠A = ∠D（已知）
∴ △ABC ≌ △DEF（SAS）
```

**3. 常见辅助线**
- 找公共边、公共角、对顶角
- 利用平行线性质
- 添加中线、高线、角平分线

### ⏰ 11:00-12:00 | 生物复习1（60分钟）

#### 复习内容：遗传学

**必须掌握的知识点：**

**1. DNA、基因、染色体的关系**
```
细胞核 → 染色体（DNA+蛋白质）→ DNA → 基因 → 控制性状
```

**记忆：**
- 染色体由DNA和蛋白质组成
- 基因是DNA上有遗传效应的片段
- 基因控制性状

---

**2. 基因型与表现型**

| 基因型 | 表现型 | 说明 |
|-------|--------|------|
| AA | 显性性状 | 显性纯合子 |
| Aa | 显性性状 | 杂合子 |
| aa | 隐性性状 | 隐性纯合子 |

**记忆：**
- 显性基因用大写字母（A）
- 隐性基因用小写字母（a）
- AA和Aa都表现显性
- aa表现隐性

---

**3. 遗传图解规范画法**

**格式：**
```
P（亲代）：  ♀AA    ×    ♂aa
               ↓
           配子：A        a
               ↓
           F1：    Aa（全部）
           表现型：显性性状
```

**要点：**
- 用P表示亲代
- 用F1表示子一代
- 用⊗表示杂交
- 写出配子
- 写出基因型和表现型

---

**4. 遗传概率**

**常见规律：**
- Aa × Aa → F1: AA(25%) : Aa(50%) : aa(25%)
  - 表现型：显性(75%) : 隐性(25%) = 3:1

- Aa × aa → F1: Aa(50%) : aa(50%)
  - 表现型：显性(50%) : 隐性(50%) = 1:1

**一、神经调节（20分钟）**

**必须掌握的知识点：**

**1. 神经系统组成**
- 中枢神经系统：脑（大脑、小脑、脑干）+ 脊髓

**2. 各部分功能**
- 大脑：感觉、运动、语言、思维中枢
- 小脑：协调运动、维持平衡
- 脑干：呼吸、心跳等生命中枢
- 脊髓：反射、传导

**3. 反射弧5个环节**
```
感受器 → 传入神经 → 神经中枢 → 传出神经 → 效应器
```

**4. 反射类型**
- 简单反射（非条件反射）：生来就有，如膝跳反射
- 复杂反射（条件反射）：后天学习，如望梅止渴

**2. 血糖调节（重点）**

**正常血糖：** 80-120 mg/dL

**调节机制：**

| 情况 | 激素 | 作用 | 结果 |
|------|------|------|------|
| 血糖↑ | 胰岛素↑ | 促进利用和储存 | 血糖↓ |
| 血糖↓ | 胰高血糖素↑ | 促进分解和转化 | 血糖↑ |

**记忆：**
- 胰岛素降低血糖

  - 消费者（动物）

**2. 食物链书写规则**

- 箭头：从被吃的指向吃的
