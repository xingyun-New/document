### ⏰ 09:00-10:30 | 数学综合复习（1.5小时）

#### 本周数学知识点总结

**知识点1：有理数运算（15分钟复习）**

**运算法则速记：**
```
加法：同号相加取相同符号，异号相加取绝对值大的符号
减法：减去一个数等于加上它的相反数
乘法：同号得正，异号得负
除法：除以一个数等于乘以它的倒数
```

---

**知识点2：整式加减（15分钟复习）**

**步骤口诀：去括号、合并同类项**

**关键点：**
- 括号前是"+"，符号不变
- 括号前是"-"，符号改变
- 同类项：字母相同，指数相同

---

**知识点3：一元一次方程（20分钟复习）**

**解方程步骤：** 去分母→去括号→移项→合并→系数化1

**应用题步骤：** 审→设→列→解→答

#### 制作生物思维导图

**第一章：动物的运动和行为**

**思维导图框架：**

```
                    动物的运动和行为
                          │
            ┌─────────────┴─────────────┐
            │                           │
        运动系统                    动物行为
            │                           │
     ┌──────┴──────┐           ┌────────┴────────┐
     │             │           │                 │
  骨骼系统     肌肉系统    先天性行为        学习行为
     │             │           │                 │
  ┌──┴──┐      ┌──┴──┐     反射、本能      条件反射、印随

```

**重点内容：**

1. **运动系统的组成**
   - 骨骼：支持和保护
   - 关节：运动的枢纽
   - 肌肉：运动的动力

2. **关节的结构**
   - 关节头
   - 关节窝
   - 关节囊
   - 关节腔
   - 关节软骨

3. **骨骼肌的特性**
   - 收缩性
   - 兴奋性

4. **先天性行为**
   - 定义：生来就有的，由遗传物质决定
   - 例子：蜘蛛织网、鸟类迁徙

5. **学习行为**
   - 定义：后天学习形成的
