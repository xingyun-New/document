### ⏰ 08:30-09:30 | 数学重点突破（60分钟）

> 🎯 **重点原因：** 今天重点攻克全等三角形证明题，这是试卷第24题（8分题）的关键！

#### 知识点：全等三角形证明技巧

**学习流程：**

**第一步：掌握证明步骤（15分钟）**

**证明步骤（5步法）：**

**1. 读题画图**
- 仔细读题，理解题意
- 准确画出图形
- 标注已知条件（用不同颜色）

**2. 寻找条件**
- 从已知条件中找出可用的边和角
- 找隐含条件：
  - 公共边
  - 公共角
  - 对顶角
  - 垂直关系（90°）
  - 平行线性质

**3. 选择判定**
- 根据找到的条件选择合适的判定定理
- SSS、SAS、ASA、AAS、HL

**4. 规范书写**
- 开头："证明：在△ABC和△DEF中"
- 写明三个条件
- 写明判定理由
- 写明结论

**5. 检查验证**
- 检查对应关系是否正确
- 检查步骤是否完整

---

**第二步：常见辅助线（20分钟）**

**辅助线的作用：**
- 构造全等三角形
- 产生新的边角关系
- 连接已知条件

**常见辅助线类型：**

**1. 延长某条边**
```
适用情况：需要构造角或边的关系
```

**2. 作中线**
```
适用情况：中点条件时
定义：连接一个顶点和对边中点
```

**3. 作高线**
```
适用情况：需要直角
定义：从一个顶点向对边作垂线
```

**4. 作角平分线**
```
适用情况：角的条件
定义：平分一个角
```

**5. 作平行线**
```
适用情况：利用平行线性质
```

**6. 连接两点**
```
适用情况：构造三角形
```

**添加辅助线的口诀：**
> 遇到中点作中线  
> 遇到垂直作高线  
> 遇到角度作角分  
> 需要平行作平行  
> 缺少条件连两点

---

**第三步：典型例题（20分钟）**

**例题1：公共边法**

```
已知：AB=AC，BD=CD
求证：△ABD≌△ACD

证明：在△ABD和△ACD中
∵ AB = AC（已知）
  BD = CD（已知）
  AD = AD（公共边）
∴ △ABD ≌ △ACD（SSS）
```

**技巧：** 找公共边！

---

**例题2：对顶角法**

```
已知：AB=CD，OB=OC，∠AOB=∠DOC（对顶角）
求证：△AOB≌△DOC

证明：在△AOB和△DOC中
∵ AB = CD（已知）
  OB = OC（已知）
  ∠AOB = ∠DOC（对顶角相等）
∴ △AOB ≌ △DOC（SAS）
```

**技巧：** 找对顶角！

---

**例题3：平行线法**

```
已知：AB∥CD，AB=CD
求证：△ABC≌△CDA

证明：在△ABC和△CDA中
∵ AB∥CD（已知）
∴ ∠BAC = ∠DCA（两直线平行，内错角相等）
又∵ AB = CD（已知）
  AC = AC（公共边）
∴ △ABC ≌ △CDA（SAS）
```

**技巧：** 利用平行线性质产生角相等！

---

**第四步：归纳总结（5分钟）**

**证明全等的常用方法：**
1. 找公共边
2. 找公共角
3. 找对顶角
4. 找平行线（内错角、同位角）
5. 找垂直关系（直角）
6. 添加辅助线

**快速判断技巧：**
- 看到三条边 → SSS
- 看到两边一角 → SAS
- 看到两角一边 → ASA或AAS
- 看到直角三角形 → HL

#### 知识点：激素调节概念

**学习流程：**

**第一步：理解激素调节（15分钟）**

**激素的定义：**
内分泌腺分泌的化学物质，通过血液运输到全身，对特定器官或细胞产生作用。

**内分泌腺：**
- 垂体
- 甲状腺
- 胸腺
- 肾上腺
- 胰岛
- 性腺

⚠️ **注意：** 内分泌腺没有导管，直接分泌到血液中！

---

**激素调节的特点：**

| 特征 | 激素调节 | 神经调节 |
|------|---------|---------|
| 作用方式 | 通过血液运输 | 通过神经冲动 |
| 反应速度 | 缓慢、持久 | 快速、准确 |
| 作用范围 | 广泛 | 精确、局部 |
| 作用时间 | 持续时间长 | 持续时间短 |

**记忆口诀：**
> 神经快速又准确，激素缓慢作用广  
> 神经局部时间短，激素持久到全身

---

**常见激素及作用：**

| 激素 | 分泌部位 | 主要作用 |
|------|---------|---------|
| 生长激素 | 垂体 | 促进生长发育 |
| 甲状腺激素 | 甲状腺 | 促进新陈代谢 |
| 胰岛素 | 胰岛 | 降低血糖 |
| 胰高血糖素 | 胰岛 | 升高血糖 |
| 性激素 | 性腺 | 促进生殖器官发育 |

---

**第二步：神经调节与激素调节的协同（10分钟）**

**协同作用：**

```
       外界刺激
           ↓
      神经系统感受
           ↓
      大脑下丘脑
           ↓
    影响内分泌腺
           ↓
     分泌激素增减
           ↓
    激素通过血液运输
           ↓
      作用于靶器官
           ↓
       产生反应
```

**举例：紧张时的身体反应**
1. 神经系统感受到紧张（快速）
2. 肾上腺分泌肾上腺素（激素）
3. 心跳加快，血压升高（持久）

**协同意义：**
- 神经调节快速启动
- 激素调节持续作用
- 共同维持内环境稳态

### ⏰ 15:15-16:15 | 生物深度学习（60分钟）

> 📝 **重点：** 血糖调节机制（必考内容！）

#### 知识点：血糖调节机制

**第一步：理解血糖的重要性（10分钟）**

**血糖的定义：**
血液中的葡萄糖，是细胞能量的主要来源。

**正常血糖浓度：**
- 空腹：80-120 mg/dL（0.8-1.2 g/L）
- 餐后：不超过180 mg/dL

**血糖的来源：**
1. 食物消化吸收（主要来源）
2. 肝糖原分解
3. 非糖物质转化（脂肪、蛋白质）

**血糖的去路：**
1. 氧化分解供能（主要去路）
2. 合成糖原储存（肝脏、肌肉）
3. 转化为脂肪等

---

**第二步：掌握血糖调节机制（30分钟）**

**血糖调节的核心：胰岛**

胰岛位于胰腺中，有两种细胞：
- **胰岛A细胞**：分泌胰高血糖素
- **胰岛B细胞**：分泌胰岛素

---

**调节机制1：血糖升高时**

```
       进餐后血糖升高
              ↓
   胰岛B细胞分泌胰岛素增加
              ↓
        胰岛素的作用：
    ①促进血糖进入细胞利用
    ②促进血糖转化为糖原储存
    ③抑制糖原分解和非糖物质转化
              ↓
         血糖浓度下降
              ↓
        恢复到正常水平
```

**胰岛素的作用（必须记住）：**
1. 促进细胞吸收和利用葡萄糖（主要作用）
2. 促进糖原合成
3. 抑制糖原分解
4. 抑制非糖物质转化为葡萄糖

**总结：胰岛素降低血糖** ↓↓↓

---

**调节机制2：血糖降低时**

```
       饥饿时血糖降低
              ↓
  胰岛A细胞分泌胰高血糖素增加
              ↓
      胰高血糖素的作用：
    ①促进肝糖原分解为葡萄糖
    ②促进非糖物质转化为葡萄糖
              ↓
         血糖浓度上升
              ↓
        恢复到正常水平
```

**胰高血糖素的作用（必须记住）：**
1. 促进肝糖原分解
2. 促进非糖物质转化为葡萄糖

**总结：胰高血糖素升高血糖** ↑↑↑

---

**两种激素的对比：**

| 激素 | 分泌细胞 | 对血糖的作用 | 作用机制 |
|------|---------|------------|---------|
| 胰岛素 | 胰岛B细胞 | 降低血糖 ↓ | 促进利用和储存 |
| 胰高血糖素 | 胰岛A细胞 | 升高血糖 ↑ | 促进分解和转化 |

**记忆技巧：**
- 胰岛素：B细胞，B=Below（降低）
- 胰高血糖素：A细胞，名字就说"高血糖"（升高）

---

**第四步：糖尿病知识（5分钟）**

**糖尿病：**
- 原因：胰岛素分泌不足或作用障碍
- 症状：血糖浓度持续过高
- 表现：多饮、多尿、多食、体重减轻（"三多一少"）
- 治疗：
  - 注射胰岛素
  - 控制饮食
  - 适量运动

⚠️ **重要：** 胰岛素是唯一能降低血糖的激素！

### ⏰ 17:15-18:00 | 道德与法治学习（45分钟）

#### 单元：第三单元（勇担社会责任）

**学习目标：**
- 理解责任的含义和来源
- 理解个人与社会的责任关系
- 学习如何承担责任
- 掌握服务社会的方式

**学习流程：**

**第一步：精读教材（25分钟）**

**主题1：责任与角色同在**

核心观点1：什么是责任
- 责任是一个人分内应该做的事情
- 责任来自对他人的承诺、职业要求、法律规定、传统习俗、公民身份等

核心观点2：每个人都有多种角色
- 在家里：子女、兄弟姐妹
- 在学校：学生、同学
- 在社会：公民、消费者
- 不同角色承担不同责任

核心观点3：角色与责任的关系
- 一个人扮演着多种角色
- 每种角色都有相应的责任
- 只有认清责任，才能更好地承担责任

**主题2：做负责任的人**

核心观点1：承担责任的意义
- 对自己负责：为自己的行为和选择负责
- 对他人负责：关爱他人，信守承诺
- 对社会负责：遵守法律，维护公共利益

核心观点2：如何做一个负责任的人
- 慎重许诺，坚决履行
- 承担过错，承担后果
- 自觉承担责任，不推诿

核心观点3：不承担责任的后果
- 损害自身利益
- 伤害他人
- 影响社会和谐
- 受到道德谴责或法律制裁

**主题3：服务社会**

核心观点1：服务社会的意义
- 实现自身价值
- 促进社会和谐
- 传递正能量
- 获得成长和快乐

核心观点2：服务社会的方式
- 参与志愿服务活动
- 关心帮助弱势群体
- 热心公益事业
- 从身边小事做起

核心观点3：青少年如何服务社会
- 积极参加社会公益活动
- 在学校做好本职工作
- 帮助有困难的同学
- 爱护公共环境和设施
- 传播文明礼仪
