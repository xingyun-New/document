**19.**（8分）

**（Ⅰ）答：**
- 第④个等式：**4×6 - 5² = -1**
- 第⑤个等式：**5×7 - 6² = -1**

**（Ⅱ）答：**
- 第n个等式：**n(n+2) - (n+1)² = -1**

**证明：**
```
n(n+2) - (n+1)²
= n² + 2n - (n² + 2n + 1)
= n² + 2n - n² - 2n - 1
= -1
```
故等式成立。

**21.**（8分）

**（Ⅰ）证明：**
```
∵ △ABC和△DCE都是等边三角形
∴ AC = BC，DC = EC，∠ACB = ∠DCE = 60°
∴ ∠ACD = ∠ACB + ∠BCD = 60° + ∠BCD
   ∠BCE = ∠DCE + ∠BCD = 60° + ∠BCD
∴ ∠ACD = ∠BCE

在△ACD和△BCE中
∵ AC = BC
   ∠ACD = ∠BCE
   DC = EC
∴ △ACD ≌ △BCE（SAS）
∴ AD = BE
即 AE = BD（证毕，注意：题目应为AD=BE或需要调整）
```

**（Ⅱ）证明：**
```
由（Ⅰ）△ACD ≌ △BCE
∴ ∠CAD = ∠CBE
∴ ∠GAC = ∠FBC

在△AGC和△BFC中
∵ ∠GAC = ∠FBC
   AC = BC
   ∠ACG = ∠BCF（公共角+60°）
∴ △AGC ≌ △BFC（ASA）
∴ AG = BF
```
