### 三、简答题（每题10分，共40分）

**28.**【遗传图解题】

（1）遗传图解：
```
P（亲代）：   Dd      ×      dd
           (高茎)         (矮茎)
             ↓
        配子：  D、d         d
             ↓       ⊗
F1（子代）：   Dd           dd
           (高茎)        (矮茎)
           比例：1    :    1
```

（2）子代中高茎：矮茎 = **1:1**

（3）子代中基因型为Dd的概率是 **1/2（或50%）**

---

**29.**【神经调节题】

（1）反射弧示意图：
```
刺激 → 感受器 → 传入神经 → 神经中枢 → 传出神经 → 效应器 → 反应
```

（2）膝跳反射属于**简单反射（非条件反射）**，神经中枢在**脊髓**。

（3）"望梅止渴"属于**复杂反射（条件反射）**，神经中枢在**大脑皮层**。

---

**30.**【激素调节题】

（1）当血糖浓度升高时：
- 胰岛分泌**胰岛素增加**
- 胰岛素促进**血糖被组织细胞利用和储存**
- 血糖浓度**降低**，恢复正常

（2）当血糖浓度降低时：
- 胰岛分泌**胰高血糖素增加**
- 胰高血糖素促进**肝糖原分解为葡萄糖**
- 血糖浓度**升高**，恢复正常

（3）糖尿病患者体内**胰岛素**分泌不足。

**31.**【生态系统题】

（1）该食物网中有**4条**食物链。

最长的食物链：**草 → 蚱蜢 → 青蛙 → 蛇 → 鹰**

（2）草属于**生产者**；鹰属于**消费者**（最高级消费者）。

（3）如果蛇的数量大量减少：
- **青蛙数量短期内会增加**（因为被蛇捕食减少）
- **鹰的数量短期内会减少**（因为食物来源减少）
