**9.** 点(-2, 5)在（  ）

### 三、解答题（共40分）

**21题（6分）参考答案：**

```
证明：在△ABD和△CDB中
     ┌ AB = CD    （已知）………………………… 1分
     │ AD = CB    （已知）………………………… 1分
     └ BD = DB    （公共边）……………………… 1分
∴ △ABD ≌ △CDB  （SSS）………………………… 2分
∴ ∠A = ∠C      （全等三角形对应角相等）…… 1分
```

---

**22题（6分）参考答案：**

```
证明：∵ ∠1 = ∠2 ………………………………… 1分

在△ABD和△CBD中
     ┌ AB = CB    （已知）………………………… 1分
     │ ∠ABD = ∠CBD（∠1 = ∠2）……………… 1分
     └ BD = BD    （公共边）……………………… 1分
∴ △ABD ≌ △CBD  （SAS）………………………… 2分
```

---

**23题（8分）参考答案：**

```
(1) 证明：在△ABD和△ACD中
     ┌ AB = AC    （已知）………………………… 1分
     │ BD = CD    （D是BC中点）……………… 1分
     └ AD = AD    （公共边）……………………… 1分
∴ △ABD ≌ △ACD  （SSS）………………………… 1分

(2) ∵ △ABD ≌ △ACD
∴ ∠ADB = ∠ADC  （全等三角形对应角相等）… 1分

∵ ∠ADB + ∠ADC = 180°（邻补角）…………… 1分
∴ ∠ADB = ∠ADC = 90° ………………………… 1分
∴ AD ⊥ BC ………………………………………… 1分
```

---

**24题（8分）参考答案：**

```
证明：∵ AB ⊥ BD，CD ⊥ BD
∴ ∠ABE = ∠DCE = 90° ………………………… 1分
∴ △ABE和△DCE都是直角三角形 ……………… 1分

∵ E是BD中点
∴ BE = DE ………………………………………… 1分

在Rt△ABE和Rt△DCE中
     ┌ AB = DC    （已知）………………………… 1分
     └ BE = DE    （E是中点）…………………… 1分

方法一（SAS）：
∠ABE = ∠DCE = 90°
∴ Rt△ABE ≌ Rt△DCE  （SAS）………………… 2分
∴ AE = CE ………………………………………… 1分

方法二（HL）：
先证AE² = AB² + BE²，CE² = CD² + DE²
∵ AB = CD，BE = DE
∴ AE = CE ………………………………………… 
```

---

**25题（6分）参考答案：**

```
解：(1) 根据勾股定理
AB² = AC² + BC²  ………………………………… 1分
AB² = 5² + 12² = 25 + 144 = 169 ……………… 1分
AB = 13cm ………………………………………… 1分

(2) 用面积法
S△ABC = ½ × AC × BC = ½ × AB × CD ……… 1分
½ × 5 × 12 = ½ × 13 × CD
30 = 6.5 × CD ……………………………………… 1分
CD = 30/6.5 = 60/13 ≈ 4.62cm ………………… 1分
```
